# 公元前19000年
- 该[Ishango骨](https://en.wikipedia.org/wiki/Ishango_bone)，可能表明已被使用的材料对象被用于简单的算术运算，它可以提供的一些知识的证据[质数](https://en.wikipedia.org/wiki/Prime_number)（虽然这是有争议的）。
# 公元前4000年
- [Quipu](https://en.wikipedia.org/wiki/Quipu)\-南美洲安第斯山脉的蒂瓦纳库人的祖先用来计数的打结的绳子。
# 公元前2500年
- 该[算盘](https://en.wikipedia.org/wiki/Abacus)，第一个已知的计算器，可能是由发明[的巴比伦人](https://en.wikipedia.org/wiki/Babylonians)作为援助简单的[算术](https://en.wikipedia.org/wiki/Arithmetic)解决此时间段。它为[位置符号](https://en.wikipedia.org/wiki/Positional_notation)和后来的[计算](https://en.wikipedia.org/wiki/Computing)发展奠定了基础。
# 公元前1770年
- 古埃及人在会计文本中首次使用[零](https://en.wikipedia.org/wiki/0_(number))。
# 公元前910年
- 指南[车](https://en.wikipedia.org/wiki/South-pointing_chariot)是[中国古代](https://en.wikipedia.org/wiki/History_of_China#Ancient_China)发明的。它是第一个已知的使用[差速齿轮的](https://en.wikipedia.org/wiki/Differential_gear)齿轮机构。战车是一辆两轮车，车上有一个指示图形，通过差速器与车轮相连。通过精心选择车轮尺寸、轨道和齿轮比，战车顶部的人物始终指向同一个方向。
# 公元前500年
- [印度语法学家](https://en.wikipedia.org/wiki/Vyakarana) [Pāṇini](https://en.wikipedia.org/wiki/P%C4%81%E1%B9%87ini)制定了[梵文语法](https://en.wikipedia.org/wiki/Sanskrit_grammar)（3959年规则），被称为[Ashtadhyayi](https://en.wikipedia.org/wiki/Ashtadhyayi)，它是高度系统化和技术性的。Pāṇini使用元[规则](https://en.wikipedia.org/wiki/Metarule)、[转换](https://en.wikipedia.org/wiki/Transformation_(mathematics))和[递归](https://en.wikipedia.org/wiki/Recursion)，其复杂程度以至于他的语法具有相当于[图灵机](https://en.wikipedia.org/wiki/Turing_machine)的计算能力。Pāṇini的工作是现代[形式语言理论](https://en.wikipedia.org/wiki/Formal_language_theory)的先驱，也是其在现代计算中使用的先驱。用于描述大多数现代编程语言的[Panini-Backus形式](https://en.wikipedia.org/wiki/Panini%E2%80%93Backus_form)也与Pāṇini的语法规则非常相似。
# 公元前200年
- [印度数学家](https://en.wikipedia.org/wiki/Indian_mathematics) [Pingala](https://en.wikipedia.org/wiki/Pingala)首先描述了[二进制数系统](https://en.wikipedia.org/wiki/Binary_number_system)，该系统现在基本上用于所有现代计算设备的设计。他还构思了类似于摩尔斯电码的[二进制代码](https://en.wikipedia.org/wiki/Binary_code)的概念。
# 公元前125年
- 该[安提凯希拉机制](https://en.wikipedia.org/wiki/Antikythera_mechanism)：发条，[模拟计算机](https://en.wikipedia.org/wiki/Analog_computer)相信已设计和建造的科林斯殖民地[雪城](https://en.wikipedia.org/wiki/Syracuse,_Sicily)。该机构包含一个[差动齿轮](https://en.wikipedia.org/wiki/Differential_gear)，能够跟踪所有当时已知的天体的相对位置。
# 9年
- [中国数学家](https://en.wikipedia.org/wiki/Chinese_mathematics)首先使用[负数](https://en.wikipedia.org/wiki/Negative_numbers)。
# 60年
- [亚历山大的英雄](https://en.wikipedia.org/wiki/Hero_of_Alexandria)做出了许多发明，包括“顺序控制”，其中机器的操作员设置机器运行，然后以确定性的方式遵循一系列指令。这基本上是第一个[程序](https://en.wikipedia.org/wiki/Computer_program)。他还在自动机领域做出了无数创新，这是[机器人技术](https://en.wikipedia.org/wiki/Robotics)发展的重要一步。
# 190年
- 在首先提及[算盘](https://en.wikipedia.org/wiki/Suanpan)（中国珠算），它被广泛使用，直到现代计算器的发明，并继续在一些文化中今天使用。
# 639年
- [印度数学家](https://en.wikipedia.org/wiki/Indian_mathematics) [Brahmagupta](https://en.wikipedia.org/wiki/Brahmagupta)是第一个描述现代[位值](https://en.wikipedia.org/wiki/Place-value) [数字系统](https://en.wikipedia.org/wiki/Numeral_system)（[印度数字系统](https://en.wikipedia.org/wiki/Hindu_numeral_system)）的人。
# 725年
- 中国发明家[梁令赞](https://en.wikipedia.org/wiki/Liang_Lingzan)建造了世界上第一座全机械钟；[水钟](https://en.wikipedia.org/wiki/Water_clocks)，其中一些非常准确，在此之前已经为人所知了几个世纪。这是一次重要的技术飞跃；一千年后制造的最早的真正计算机使用基于时钟的技术。
# 820年
- [波斯数学家](https://en.wikipedia.org/wiki/Islamic_mathematics)，[花拉子米](https://en.wikipedia.org/wiki/Mu%E1%B8%A5ammad_ibn_M%C5%ABs%C4%81_al-Khw%C4%81rizm%C4%AB)，描述现代的雏形[代数](https://en.wikipedia.org/wiki/Algebra)的名字是从他的书中得到的[铝基塔AL-FImuḫtaṣar人ḥisāb-贾布尔WA-1-muqābala](https://en.wikipedia.org/wiki/The_Compendious_Book_on_Calculation_by_Completion_and_Balancing)。这个词[算法](https://en.wikipedia.org/wiki/Algorithm)，距离Al-花拉子米的拉丁名称派生Algoritmi。
# 850年
- [阿拉伯数学家](https://en.wikipedia.org/wiki/Islamic_mathematics)，[肯迪](https://en.wikipedia.org/wiki/Al-Kindi)（Alkindus），当时的先驱[密码](https://en.wikipedia.org/wiki/Cryptography)。他在AManuscriptonDecipheringCryptographicMessages中给出了第一个已知的[密码分析](https://en.wikipedia.org/wiki/Cryptanalysis)记录解释。特别是，他因开发了频率分析方法而受到赞誉，由此可以分析和利用字母出现频率的变化来破解加密密码（即通过频率分析进行密码分析）。本书还介绍了密码分析、加密、某些加密的密码分析以及阿拉伯语字母和字母组合的统计分析的方法。
- 该[班·马萨](https://en.wikipedia.org/wiki/Ban%C5%AB_M%C5%ABs%C4%81)兄弟，在他们的[巧妙手法的书](https://en.wikipedia.org/wiki/Book_of_Ingenious_Devices)，发明了“已知最早的机械[乐器](https://en.wikipedia.org/wiki/Musical_instrument)”，在这种情况下[hydropowered](https://en.wikipedia.org/wiki/Hydropower) [器官](https://en.wikipedia.org/wiki/Organ_(music))，其自动播放互换缸。直到19世纪下半叶，这种“表面带有凸起销钉的圆柱体仍然是机械地制作和再现音乐的基本装置”。他们还发明了一种[自动](https://en.wikipedia.org/wiki/Automaton) [长笛演奏](https://en.wikipedia.org/wiki/Flute)器，这似乎是第一台[可编程](https://en.wikipedia.org/wiki/Computer_programming)机器。
# 1000年
- [AbūRayhānal-Bīrūnī](https://en.wikipedia.org/wiki/Ab%C5%AB_Rayh%C4%81n_al-B%C4%ABr%C5%ABn%C4%AB)发明了[Planisphere](https://en.wikipedia.org/wiki/Planisphere)，一种[模拟计算机](https://en.wikipedia.org/wiki/Analog_computer)。他还发明了第一[机械](https://en.wikipedia.org/wiki/Mechanical_engineering) [阴阳历](https://en.wikipedia.org/wiki/Lunisolar_calendar)其中使用的[齿轮系](https://en.wikipedia.org/wiki/Gear_train)和八个[齿轮](https://en.wikipedia.org/wiki/Gear)\-wheels。这是一个固定有线知识处理[机器](https://en.wikipedia.org/wiki/Machine)的早期例子。
# 1015年
- [阿拉伯天文学家](https://en.wikipedia.org/wiki/Islamic_astronomy)，安达卢斯的[AbūIshāqIbrāhīmal-Zarqālī](https://en.wikipedia.org/wiki/Ab%C5%AB_Ish%C4%81q_Ibr%C4%81h%C4%ABm_al-Zarq%C4%81l%C4%AB)(Arzachel)发明了赤道仪，这是一种机械模拟计算机设备，用于无需计算即可找到月球、太阳和行星的经度和位置，使用几何图形模型来表示天体的平均和异常位置。
# 1150年
- [阿拉伯天文学家](https://en.wikipedia.org/wiki/Islamic_astronomy)，[贾比尔·伊本·阿弗拉](https://en.wikipedia.org/wiki/Jabir_ibn_Aflah)（迦别），发明了[Torquetum](https://en.wikipedia.org/wiki/Torquetum)，一个观察仪器和机械[模拟计算机](https://en.wikipedia.org/wiki/Analog_computer)使用之间变换装置[球形坐标系统](https://en.wikipedia.org/wiki/Spherical_coordinate_system)。它被设计用来获取和转换在三组坐标中进行的测量：[地平线](https://en.wikipedia.org/wiki/Horizon)、[赤道](https://en.wikipedia.org/wiki/Equator)和[黄道](https://en.wikipedia.org/wiki/Ecliptic)。
# 1206年
- [阿拉伯工程师](https://en.wikipedia.org/wiki/Inventions_in_the_Islamic_world)，[加扎利](https://en.wikipedia.org/wiki/Al-Jazari)，发明了无数的[自动机](https://en.wikipedia.org/wiki/Automaton)和提出了许多其他技术创新。其中之一是一个设计[的可编程](https://en.wikipedia.org/wiki/Program_(machine)) [人形](https://en.wikipedia.org/wiki/Humanoid_robot)形[时装模特](https://en.wikipedia.org/wiki/Mannequin)：这似乎是第一次认真，科学的（而不是神奇）的计划[机器人](https://en.wikipedia.org/wiki/Robot)。他还发明了“[城堡钟](https://en.wikipedia.org/wiki/Castle_clock)”，一种被认为是最早的可编程模拟计算机的[天文钟](https://en.wikipedia.org/wiki/Astronomical_clock)。它显示了黄道带、太阳和月球的轨道、一个 [月牙](https://en.wikipedia.org/wiki/Lunar_phase)形[指针](https://en.wikipedia.org/wiki/Pointer_(computer_programming))旅游跨越造成自动门打开每小时网关，和五个[机器人](https://en.wikipedia.org/wiki/Robot)敲击时通过杠杆由操作谁玩音乐的音乐家[凸轮轴](https://en.wikipedia.org/wiki/Camshaft)连接到[水车](https://en.wikipedia.org/wiki/Water_wheel)。白天和黑夜的长度可以每天重新编程，以考虑全年白天和黑夜的长度变化。
# 1235年
- [伊斯法罕的](https://en.wikipedia.org/wiki/Isfahan)波斯天文学家AbiBakr根据AbūRayhānal-Bīrūnī的机械日历模拟计算机的设计，发明了一种带有齿轮日历机芯的黄铜[星盘](https://en.wikipedia.org/wiki/Astrolabe)。AbiBakr的齿轮星盘使用一组齿轮，是现存最古老的完整机械齿轮机器。
# 1300年
- [RamonLlull](https://en.wikipedia.org/wiki/Ramon_Llull)发明了LullianCircle：一种通过逻辑组合计算哲学问题（在这种情况下，与基督教有关）答案的概念机器。这个想法在几个世纪后被[莱布尼茨](https://en.wikipedia.org/wiki/Gottfried_Leibniz)采纳，因此成为计算和[信息科学的基本](https://en.wikipedia.org/wiki/Information_science)要素之一。
# 1412年
- [艾哈迈德沙·阿尔·坎尔基香迪](https://en.wikipedia.org/wiki/Ahmad_al-Qalqashandi)给出的列表[密码](https://en.wikipedia.org/wiki/Cipher)在他Subh阿尔·阿奥沙其中包括[替代](https://en.wikipedia.org/wiki/Substitution_cipher)和[换位](https://en.wikipedia.org/wiki/Transposition_cipher)，并在第一时间，用多重替换为每一个密码[明文](https://en.wikipedia.org/wiki/Plaintext)字母。他还给出了[密码分析的](https://en.wikipedia.org/wiki/Cryptanalysis)说明和工作示例，包括使用[字母频率](https://en.wikipedia.org/wiki/Letter_frequencies)表和不能在一个单词中一起出现的字母组。
# 1416年
- [Jamshīdal-Kāshī](https://en.wikipedia.org/wiki/Jamsh%C4%ABd_al-K%C4%81sh%C4%AB)发明了合相板，这是一种[模拟计算机](https://en.wikipedia.org/wiki/Analog_computer)仪器，用于确定一天中[行星合相](https://en.wikipedia.org/wiki/Planetary_conjunction)发生的时间，并用于执行[线性插值](https://en.wikipedia.org/wiki/Linear_interpolation)。他还发明了一种机械的“行星电脑”，他叫区的板块，这可能图形解决了许多行星的问题，包括真实位置在预测[经度](https://en.wikipedia.org/wiki/Longitude)太阳和月亮的，和[行星](https://en.wikipedia.org/wiki/Planet);太阳、月亮和行星的[纬度](https://en.wikipedia.org/wiki/Latitude)；和[黄道](https://en.wikipedia.org/wiki/Ecliptic)太阳的。该仪器还集成了[alhidade](https://en.wikipedia.org/wiki/Alhidade)和[尺子](https://en.wikipedia.org/wiki/Ruler)。
# 1450年
- [天文学和数学的喀拉拉学校](https://en.wikipedia.org/wiki/Kerala_school_of_astronomy_and_mathematics)在[南印度](https://en.wikipedia.org/wiki/South_India)发明了[浮点](https://en.wikipedia.org/wiki/Floating-point)数字系统。
# 1493年
- [列奥纳多·达·芬奇(LeonardodaVinci)](https://en.wikipedia.org/wiki/Leonardo_da_Vinci)绘制了一个由互锁齿轮组成的设备图纸，可以将其解释为能够进行加法和减法的[机械计算器](https://en.wikipedia.org/wiki/Mechanical_calculator)。受该计划启发的工作模型建于1968年，但莱昂纳多是否真的想到了计算器仍然存在争议。达芬奇还为机械人制定了计划：早期的[机器人](https://en.wikipedia.org/wiki/Robot)设计。
# 1614年
- 苏格兰人[约翰·纳皮尔(JohnNapier)](https://en.wikipedia.org/wiki/John_Napier)重新发明了一种对数形式和巧妙的可移动杆系统（1617年，被称为纳皮尔的杆或[纳皮尔的骨头](https://en.wikipedia.org/wiki/Napier%27s_bones)）。这些杆基于[格子或gelosia乘法](https://en.wikipedia.org/wiki/Lattice_multiplication)算法，允许操作员通过移动杆并将它们放置在特殊构造的板上来乘、除和计算平方根和立方根。
# 1622年
- [WilliamOughtred](https://en.wikipedia.org/wiki/William_Oughtred)根据JohnNapier开发的对数开发了[计算尺](https://en.wikipedia.org/wiki/Slide_rule)。
# 1623年
- 德国[博学者](https://en.wikipedia.org/wiki/Polymath) [威廉·席卡德](https://en.wikipedia.org/wiki/Wilhelm_Schickard)(WilhelmSchickard)在他寄给约翰内斯·开普勒(JohannesKepler)的两封信上画了一个他称之为计算时钟的装置。一个是在1623年，另一个是在1624年。1624年正在建造的机器后来被一场大火烧毁，他决定放弃他的项目。这台机器直到1957年这两个字母被发现时才为世人所知。一些复制品建于1961年。这台机器对机械计算器的发展没有影响。
# 1642年
- 在法国，法国博学者[布莱斯·帕斯卡](https://en.wikipedia.org/wiki/Blaise_Pascal)发明了机械计算器。被称为机器算术、帕斯卡的计算器和最终的[帕斯卡林](https://en.wikipedia.org/wiki/Pascaline)，它在1645年的公开介绍开始了机械计算器的开发，首先是在欧洲，然后是世界其他地方。它是第一台具有受控运载机构的机器。Pascal在发布他的第一台机器之前制造了50台原型机（最终制造了20台机器）。帕斯卡林启发了[GottfriedLeibniz](https://en.wikipedia.org/wiki/Gottfried_Leibniz)（1671年）、[ThomasdeColmar](https://en.wikipedia.org/wiki/Thomas_de_Colmar)（1820年）和[DorrE.Felt](https://en.wikipedia.org/wiki/Dorr_E._Felt)（1887年）的作品。
# 1666年
- 在英国，英国的[塞缪尔·莫兰](https://en.wikipedia.org/wiki/Samuel_Morland)爵士（SirSamuelMorland，1625-1695年）生产了一种非十进制加法机，适用于[英国货币](https://en.wikipedia.org/wiki/Pound_sterling)。它没有使用进位机制，而是在辅助拨号上注册进位，用户从中重新输入它们作为加数。
# 1672年
- 在德国，德国[数学家](https://en.wikipedia.org/wiki/Mathematician)，[莱布尼兹](https://en.wikipedia.org/wiki/Gottfried_Leibniz)开始设计一种机器，其事半功倍，“[阶梯清算者](https://en.wikipedia.org/wiki/Stepped_Reckoner)”。它可以将最多5位和12位的数字相乘，得到16位的结果。制造了两台机器，一台在1694年（1879年在阁楼中被发现），另一台在1706年。
# 1685年
- 在德国，戈特弗里德莱布尼茨在一篇题为“在几乎没有分界线的情况下不仅可以执行加减法和乘法的算术机器”的文章中，[戈特弗里德莱布尼茨](https://en.wikipedia.org/wiki/Gottfried_Leibniz)描述了一种使用[带有活动齿的轮子](https://en.wikipedia.org/wiki/Pinwheel_calculator)的机器，当它与帕斯卡林耦合时，可以执行所有四个数学运算。没有证据表明莱布尼茨曾经建造过这种风车机器。
# 1709年
- 在意大利，[GiovanniPoleni](https://en.wikipedia.org/wiki/Giovanni_Poleni)是第一个使用[风车设计](https://en.wikipedia.org/wiki/Pinwheel_calculator)构建计算器的人。它是用木头做的，做成一个计算钟的形状。
# 1726年
- 在英国，[乔纳森·斯威夫特](https://en.wikipedia.org/wiki/Jonathan_Swift)（JonathanSwift）在他的《[格列佛游记](https://en.wikipedia.org/wiki/Gulliver%27s_Travels)》中（讽刺地）描述了一台机器（“引擎”）。“引擎”由一个木制框架和包含词性的木块组成。当发动机的40个杠杆同时转动时，机器显示语法句子片段。
# 1774年
- 在德国，[菲利普·马特乌斯·哈恩](https://en.wikipedia.org/wiki/Philipp_Matth%C3%A4us_Hahn)(PhilippMatthäusHahn)在现在的德国制造了一款成功的便携式计算器，能够执行所有四种数学运算。
# 1775年
- 在英国，[查尔斯·斯坦霍普(CharlesStanhope)，](https://en.wikipedia.org/wiki/Charles_Stanhope,_3rd_Earl_Stanhope)英格兰第三代伯爵斯坦霍普(CharlesStanhope)，设计并建造了一个成功的乘法计算器，类似于莱布尼茨(Leibniz)的计算器。
# 1786年
- 在德国，[JHMüller](https://en.wikipedia.org/wiki/J._H._M%C3%BCller)是Hessian军队的一名工程师，他首先想到了[差分机](https://en.wikipedia.org/wiki/Difference_engine)的想法（第一次书面提及差分机的基本原理可以追溯到1784年）。
# 1804年
- 在法国，[Joseph-MarieJacquard](https://en.wikipedia.org/wiki/Joseph-Marie_Jacquard)开发了[Jacquard织机](https://en.wikipedia.org/wiki/Jacquard_loom)，一种由[穿孔卡片](https://en.wikipedia.org/wiki/Punched_card)控制的自动织机。
# 1820年
- 在法国，[CharlesXavierThomasdeColmar](https://en.wikipedia.org/wiki/Charles_Xavier_Thomas_de_Colmar)发明了“[算术计](https://en.wikipedia.org/wiki/Arithmometer)”，经过三十多年的发展，它于1851年成为第一台批量生产的机械计算器。通过对结果使用可移动累加器，操作员可以快速有效地执行[长乘法](https://en.wikipedia.org/wiki/Multiplication_algorithm)和除法运算。这台机器基于Pascal和Leibniz的早期作品。
# 1822年
- 在英国，[查尔斯·巴贝奇](https://en.wikipedia.org/wiki/Charles_Babbage)设计了他的第一台机械计算机，这是用于制表多项式的小数[差分引擎](https://en.wikipedia.org/wiki/Difference_engine)的第一个原型。
# 1831年
- 在意大利，[GiovanniPlana](https://en.wikipedia.org/wiki/Giovanni_Plana)设计了一台[万年历机](https://en.wikipedia.org/wiki/Cappella_dei_Mercanti_(Turin)#Perpetual_calendar)，它可以精确计算超过4000年的日历，计算闰年和昼长变化。
# 1832年
- 在俄罗斯，[SemenKorsakov](https://en.wikipedia.org/wiki/Semen_Korsakov)提议使用[穿孔卡片](https://en.wikipedia.org/wiki/Punched_card)来存储和搜索信息。他设计了几台机器来展示他的想法，包括所谓的线性内窥镜。
- 在英国，巴贝奇和[约瑟夫克莱门特](https://en.wikipedia.org/wiki/Joseph_Clement)制作了他的原型段[差发动机](https://en.wikipedia.org/wiki/Difference_engine)，，其在6个位数字和二阶差（即，它可以制成表二次多项式）操作。完整的引擎本来是房间大小，计划在大约20位数字的六阶差分和30位数字的三阶差分上运行。每个加法都分两个阶段完成，第二个阶段处理第一个阶段产生的任何进位。输出数字将被冲压到一块软金属板上，用它可能已经制成了印刷板。但是遇到了各种各样的困难，最终完成的只是这件原型作品。
# 1833年
- 在英国，巴贝奇构思并开始设计他的十进制“[分析引擎](https://en.wikipedia.org/wiki/Analytical_Engine)”。它的[程序](https://en.wikipedia.org/wiki/Computer_program)以穿孔卡片的形式[存储](https://en.wikipedia.org/wiki/Computer_data_storage)在[只读存储器](https://en.wikipedia.org/wiki/Read-only_memory)中。巴贝奇多年来一直致力于设计，但在大约1840年之后，设计变化似乎很小。这台机器会在40位数字上运行；'mill'(CPU)将有2个主累加器和一些用于特定目的的辅助累加器，而'store'（内存）将保存一千个50位数字。本来会有几个打孔卡读卡器，用于程序和数据;卡片被链接起来，每条链条的运动都是可逆的。机器会执行条件跳转。也会有一种[微编码](https://en.wikipedia.org/wiki/Microcode)形式：[指令](https://en.wikipedia.org/wiki/Instruction_(computer_science))的含义取决于金属螺柱在开槽桶中的位置，称为“控制桶”。所设想的机器能够在3秒内完成加法运算，在2-4分钟内完成乘法或除法运算。它由[蒸汽机提供动力](https://en.wikipedia.org/wiki/Steam_engine)。最后，实际建造的部件不超过几个。
# 1835年

# 1842年
- 在法国，TimoleonMaurel为[Arithmaurel](https://en.wikipedia.org/wiki/Arithmaurel)申请了专利，这是一种具有非常直观的用户界面的机械计算器，特别适用于乘法和除法，因为只要输入操作数，结果就会显示出来。它于1849年在巴黎举行的法国国家展览会上获得了金牌。不幸的是，其设计的复杂性和脆弱性使其无法生产。
- 在英国，巴贝奇[差分引擎的](https://en.wikipedia.org/wiki/Difference_engine)建造作为正式项目被取消。成本超支相当可观（花费了17,470英镑，2004年的资金约为1,000,000英镑）。
# 1843年
- 在瑞典，[PerGeorgScheutz](https://en.wikipedia.org/wiki/Per_Georg_Scheutz)和他的儿子Edvard制作了一个5位数字和带有打印机的[差分引擎](https://en.wikipedia.org/wiki/Difference_engine)的三阶模型；瑞典政府于1851年同意为他们的下一步发展提供资金。
# 1846年
- 在英国，巴贝奇开始研究改进的[差分引擎](https://en.wikipedia.org/wiki/Difference_engine)（2号差分引擎），到1849年产生了一套完全执行的计划。这台机器本来可以处理7阶差分和31位数字，但没有人发现要付费才能建造它。1989-1991年，伦敦科学博物馆的一个团队确实根据幸存的计划建造了一个。他们使用现代方法构建组件，但公差并不比Clement所能提供的更好……并且，经过一些修补和细节调试，他们发现机器工作正常。2000年，打印机也完成了。
# 1847年
- 在英国，英国数学家[GeorgeBoole](https://en.wikipedia.org/wiki/George_Boole)开发了二进制代数（[Booleanalgebra](https://en.wikipedia.org/wiki/Boolean_algebra_(logic))），它在大约一个世纪后开始广泛应用于二进制计算机的设计和运算。见1939年。
# 1851年
- 在法国，经过30年的发展，[ThomasdeColmar](https://en.wikipedia.org/wiki/Thomas_de_Colmar)通过开始制造大大简化的[算术计](https://en.wikipedia.org/wiki/Arithmometer)（发明于1820年），开创了机械计算器行业。除了30年后开始的克隆之外，它是40年来世界上唯一可用的计算机（[DorrE.Felt](https://en.wikipedia.org/wiki/Dorr_Felt)从1887年到1890年仅售出一百台[计算机](https://en.wikipedia.org/wiki/Comptometer)和几台[计算机](https://en.wikipedia.org/wiki/Comptograph)）。它的简单性使其成为迄今为止最可靠的计算器。这是一台大机器（一个20位的算术仪足够长，可以占据桌面的大部分）。尽管该算法直到1915年才生产，但直到二战开始，二十家欧洲公司制造了其设计的改进克隆；他们是Burkhardt、Layton、Saxonia、Gräber、Peerless、Mercedes-Euklid、XxX、Archimedes等等……。
# 1853年
- 在瑞典，令巴贝奇高兴的是，[Scheutzes](https://en.wikipedia.org/wiki/Per_Georg_Scheutz)完成了第一个全尺寸[差分引擎](https://en.wikipedia.org/wiki/Difference_engine)，他们称之为制表机。它对15位数字和4阶差分进行运算，并像巴贝奇那样产生打印输出。1859年，伦敦的[BryanDonkin](https://en.wikipedia.org/wiki/Bryan_Donkin)公司按照相同的设计制造了第二台机器。
# 1856年
- 在美国，第一台制表机（见1853年）由[纽约奥尔巴尼](https://en.wikipedia.org/wiki/Albany,_New_York)的达德利天文台购买，第二台于1857年由英国政府订购。奥尔巴尼机器被用来制作一套天文台；但是天文台的馆长因为这次奢侈的购买被解雇了，这台机器再也没有认真使用过，最终进了博物馆。第二台机器有很长的使用寿命。
# 1859年
- 在瑞典，[MartinWiberg](https://en.wikipedia.org/wiki/Martin_Wiberg)生产了一种重新设计[的类似差分引擎的](https://en.wikipedia.org/wiki/Difference_engine)机器，用于准备利率（1860年首次出版）和[对数](https://en.wikipedia.org/wiki/Logarithm)表（1875年首次出版）。
# 1866年
- 在英国，第一台实用的逻辑机器（[逻辑算盘](https://en.wikipedia.org/wiki/Logical_abacus)）是由[威廉·斯坦利·杰文斯](https://en.wikipedia.org/wiki/William_Stanley_Jevons)(WilliamStanleyJevons)建造的。
# 1871年
- 在英国，巴贝奇制作了[分析引擎](https://en.wikipedia.org/wiki/Analytical_Engine)磨机和打印机的原型部分。
# 1878年
- 在西班牙，居住在纽约市的RamónVerea发明了一个带有内部乘法表的计算器；这比移动马车或当时的其他数字方法要快得多。然而，他对将其投入生产并不感兴趣。似乎他只是想表明西班牙人可以像美国人一样进行发明。
- 在英国，一个委员会调查了完成[分析引擎](https://en.wikipedia.org/wiki/Analytical_Engine)的可行性，并得出结论，现在巴贝奇已经死了，这是不可能的。除了极少数人之外，该项目在很大程度上被遗忘了。[霍华德艾肯](https://en.wikipedia.org/wiki/Howard_Aiken)是一个明显的例外。
# 1884年
- 在美国，芝加哥的DorrFelt开发了他的[Comptometer](https://en.wikipedia.org/wiki/Comptometer)。这是第一个通过按键输入操作数而不是例如拨入的计算器。这是可行的，因为Felt发明了一种进位机制，其速度足够快，可以在按键从按下状态返回时执行操作。Felt和Tarrant于1887年开始合作制造计算器。
# 1886年
- 在美国，巴尔的摩卫生部首次使用[HermanHollerith](https://en.wikipedia.org/wiki/Herman_Hollerith)制表系统。
# 1887年
- 在美国，[HermanHollerith](https://en.wikipedia.org/wiki/Herman_Hollerith)提交了一项[集成制表器](https://en.wikipedia.org/wiki/Unit_record_equipment#Timeline)的专利申请（1890年获得批准），它可以添加[打孔卡](https://en.wikipedia.org/wiki/Punched_card)上编码的数字。1889年，陆军外科医生办公室首次记录使用该设备。1896年Hollerith引入了改进模型。
# 1889年
- 在美国，[DorrFelt](https://en.wikipedia.org/wiki/Dorr_Felt)发明了第一台打印台计算器。
# 1890年
- 在美国瑞典俄罗斯，比算术计更紧凑的乘法计算器进入批量生产。该设计是美国的[FrankS.Baldwin](https://en.wikipedia.org/wiki/Frank_S._Baldwin)和居住在俄罗斯的瑞典人[WillgodtTheophilOdhner](https://en.wikipedia.org/wiki/Willgodt_Theophil_Odhner)的独立发明，或多或少是同时发明的。凹槽鼓被“可变齿齿轮”设计所取代：一个带有径向销钉的圆盘，可以从中伸出或缩回。
- 在美国，在[1880年美国人口普查](https://en.wikipedia.org/wiki/1880_United_States_Census)采取了7年完整的，因为所有的处理一直用手从期刊表来完成。不断增长的人口表明，到[1890年人口普查时](https://en.wikipedia.org/wiki/1890_United_States_Census)，数据处理所需的时间将比下次人口普查前的10年更长——因此举行了一场竞赛以寻找更好的方法。它是由人口普查部员工[HermanHollerith](https://en.wikipedia.org/wiki/Herman_Hollerith)赢得的，他后来[创立](https://en.wikipedia.org/wiki/Tabulating_Machine_Company)了制表机公司，后来成为[IBM](https://en.wikipedia.org/wiki/International_Business_Machines)。他发明了将数据记录在可以被机器读取的介质上。以前使用机器可读媒体是为了控制（[自动机](https://en.wikipedia.org/wiki/Automaton)、[钢琴卷](https://en.wikipedia.org/wiki/Piano_roll)、[织布机）](https://en.wikipedia.org/wiki/Jacquard_loom),...)，而不是数据。“在使用纸带进行了一些初步试验后，他决定使用[穿孔卡片](https://en.wikipedia.org/wiki/Punched_card)……”他的机器使用机械[继电器](https://en.wikipedia.org/wiki/Relay)来增加机械计数器。这种方法用于1890年的人口普查。1880年人口普查的许多变化的净效果：更多的人口、要收集的数据项、人口普查局的人数、预定的出版物以及使用Hollerith的机电制表机，减少了处理人口普查所需的时间从[1880年人口普查的](https://en.wikipedia.org/wiki/U.S._Census,_1880)八年到1890年人口普查的六年。本发明的灵感来自Hollerith在美国西部旅行期间对铁路列车员的观察;他们用打票的方式对乘客（高大、秃顶、男性）进行了粗略的描述。
# 1891年
- 在美国，圣路易斯的WilliamS.Burroughs于1885年发明了一种类似于Felt的机器（见1884年），但与Comptometer不同的是，它是一种“钥匙设置”机器，仅在拉动曲柄手柄后处理每个数字。这台机器的真正制造始于1891年，尽管Burroughs于1886年创立了他的[美国算术公司](https://en.wikipedia.org/wiki/American_Arithmometer_Company)（后来成为[Burroughs公司](https://en.wikipedia.org/wiki/Burroughs_Corporation)，现在称为[Unisys](https://en.wikipedia.org/wiki/Unisys)）。

# 1906年
- 在英国，查尔斯的儿子亨利·巴贝奇在RWMunro公司的帮助下，用他父亲的[分析](https://en.wikipedia.org/wiki/Analytical_Engine)机完成了“磨坊”，以证明它是可行的。确实如此。未生产整机。

# 1930年
- 在美国，[VannevarBush](https://en.wikipedia.org/wiki/Vannevar_Bush)建造了一个能够求解微分方程的部分电子[微分分析仪](https://en.wikipedia.org/wiki/Differential_analyser)。
- 在英国，威尔士物理学家[CE韦恩-威廉姆斯](https://en.wikipedia.org/wiki/C._E._Wynn-Williams)，在[英国剑桥](https://en.wikipedia.org/wiki/Cambridge,_England)，使用的环[闸流管](https://en.wikipedia.org/wiki/Thyratron)的管来构造所计数发射的二进制数字计数器[α粒子](https://en.wikipedia.org/wiki/Alpha_particle)。
# 1931年
- 在奥地利，[哥德尔](https://en.wikipedia.org/wiki/Kurt_G%C3%B6del)的[维也纳大学](https://en.wikipedia.org/wiki/Vienna_University)，奥地利，基于算术运算的通用形式语言发表了一篇论文。他用它来编码任意的形式陈述和证明，并表明传统数学等[形式系统](https://en.wikipedia.org/wiki/Formal_system)要么在某种意义上不一致，要么包含无法证明但真实的陈述。这个结果通常被称为理论计算机科学的基本结果。

# 1934年
- 在日本,苏联,美国，1934年到1937年，[NEC](https://en.wikipedia.org/wiki/NEC)工程师[AkiraNakashima](https://en.wikipedia.org/wiki/Akira_Nakashima)、[ClaudeShannon](https://en.wikipedia.org/wiki/Claude_Shannon)和ViktorShetakov发表了一系列介绍[开关电路理论](https://en.wikipedia.org/wiki/Switching_circuit_theory)的论文。

# 1936年
- 在英国，[阿兰·图灵](https://en.wikipedia.org/wiki/Alan_Turing)的[剑桥大学](https://en.wikipedia.org/wiki/University_of_Cambridge)，英格兰，发表了“可计算数字”一文其重新[哥德尔](https://en.wikipedia.org/wiki/Kurt_G%C3%B6del)的结果（见相关工作[邱奇](https://en.wikipedia.org/wiki/Alonzo_Church)）。他的论文解决了著名的“[Entscheidungsproblem](https://en.wikipedia.org/wiki/Entscheidungsproblem)”，在论文中通过推理（作为一种数学设备）关于一台简单的理论计算机（今天称为[图灵机](https://en.wikipedia.org/wiki/Turing_machine)）来寻求解决方案。在许多方面，这种设备比哥德尔基于算术的通用形式系统更方便。
# 1937年
- 在美国，纽约市贝尔电话实验室（贝尔实验室）的GeorgeStibitz使用继电器构建了一个演示1位二进制加法器。这是最早的二进制计算机之一，虽然在这个阶段它只是一台演示机；改进继续导致了1940年1月的[复数计算器](https://en.wikipedia.org/wiki/Complex_Number_Calculator)。
- 在美国，[ClaudeE.Shannon](https://en.wikipedia.org/wiki/Claude_E._Shannon)在麻省理工学院的硕士论文中发表了一篇关于使用继电器实现符号逻辑的论文。
# 1938年
- 在德国，柏林的KonradZuse完成了“[Z1](https://en.wikipedia.org/wiki/Z1_(computer))”，这是第一台机械二进制可编程计算机。它基于布尔代数并具有现代机器的一些基本成分，使用二进制系统和浮点运算。Zuse1936年的专利申请(Z23139/GMDNr.005/021)还提出了一种“冯诺依曼”架构（大约在1945年重新发明），其中程序和数据可在存储中修改。该机器最初被称为“V1”，但战后追溯改名，以避免与V-1飞弹混淆.它适用于浮点数（7位指数、16位尾数和符号位）。内存使用滑动金属部件来存储16个这样的数字，运行良好；但算术单元不太成功，偶尔会遇到某些机械工程问题。该程序是从废弃的35毫米电影胶片上打孔中读取的。数据值可以从数字键盘输入，输出显示在电灯上。这台机器不是通用计算机（即[图灵完备](https://en.wikipedia.org/wiki/Turing_complete)），因为它缺乏循环功能。
# 1939年
- 在美国，[威廉·休利特](https://en.wikipedia.org/wiki/William_Reddington_Hewlett)和[大卫·帕卡德](https://en.wikipedia.org/wiki/David_Packard)在加利福尼亚州帕洛阿尔托帕卡德的车库中成立了[惠普公司](https://en.wikipedia.org/wiki/Hewlett-Packard_Company)，初期投资为538美元（相当于2020年的9,891美元）；这被认为是硅谷的象征性成立。惠普将发展成为当今世界上最大的科技公司之一。
- 在德国，[HelmutSchreyer](https://en.wikipedia.org/wiki/Helmut_Schreyer)使用真空管完成了一个原型10位加法器，以及一个使用霓虹灯的原型存储器。
- ## 11月
	- 在美国，[JohnVincentAtanasoff](https://en.wikipedia.org/wiki/John_Vincent_Atanasoff)和爱荷华州立大学（现爱荷华州立大学）的研究生[CliffordBerry](https://en.wikipedia.org/wiki/Clifford_Berry)完成了一个原型16位加法器。这是第一台使用真空管进行计算的机器。
# 1940年
- 在美国，在[贝尔实验室](https://en.wikipedia.org/wiki/Bell_Labs)，塞缪尔·威廉姆斯和[乔治·斯蒂比茨](https://en.wikipedia.org/wiki/George_Stibitz)完成了一个可以对[复数](https://en.wikipedia.org/wiki/Complex_number)进行运算的计算器，并将其命名为“[复数计算器”。](https://en.wikipedia.org/wiki/Complex_Number_Calculator)';它后来被称为“I型继电器计算器”。它使用电话交换部件进行逻辑：450个继电器和10个纵横开关。数字以“加3BCD”表示；也就是说，对于每个十进制数字，0用二进制0011表示，1用0100表示，依此类推，直到1100表示9；该方案比直接BCD需要更少的继电器。计算器并没有要求用户到机器前使用它，而是在建筑物的不同地方以电传打字机的形式提供了三个远程键盘。一次只能使用一个，并且输出会自动显示在同一个上。1940年9月9日，在新罕布什尔州汉诺威的[达特茅斯学院](https://en.wikipedia.org/wiki/Dartmouth_College)安装了一台电传打字机，连接到纽约，参加会议的人可以远程使用机器。
- 在德国，[KonradZuse](https://en.wikipedia.org/wiki/Konrad_Zuse)完成了“[Z2](https://en.wikipedia.org/wiki/Z2_(computer))”（原为“V2”），它将Z1现有的机械记忆单元与使用继电器逻辑的新算术单元相结合。与Z1一样，Z2缺乏循环功能。该计划在1939年起草Zuse时中断了一年，但在他获释后继续进行。1940年，ZuseZ2在柏林阿德勒斯霍夫向德国航空研究所（“德国航空实验室”）的观众展示。
# 1941年
- ## 5月
	- ### 11日
		- 在德国，现在，在DVL（德国航空研究所）的有限支持下，[KonradZuse](https://en.wikipedia.org/wiki/Konrad_Zuse)完成了“**[Z3](https://en.wikipedia.org/wiki/Z3_(computer))”**（原为“V3”）：第一台可操作的可编程计算机。对[CharlesBabbage](https://en.wikipedia.org/wiki/Charles_Babbage)的非功能性设备的一项重大改进是使用[Leibniz](https://en.wikipedia.org/wiki/Gottfried_Leibniz)的二进制系统（巴贝奇和其他人尝试构建十进制可编程计算机失败）。Zuse的机器还具有浮点数，具有7位指数、14位尾数（除非数字为0，否则自动带有“1”位前缀）和一个符号位。内存保存了64个这样的单词，因此需要1400多个继电器；算术和控制单元有1200多个。它还具有并行加法器。程序、输入和输出的实现与上述Z1相同。虽然条件跳转不可用，但事实证明，Zuse的Z3原则上能够作为通用计算机运行。机器每秒可以做3-4次加法，乘法需要3-5秒。Z3于1943年在盟军轰炸柏林期间被摧毁，对美国和英国的计算机技术没有影响。
# 1942年
- 在美国，Atanasoff和Berry完成了一个用于求解联立线性方程组的专用计算器，后来称为“ABC”（“[Atanasoff–Berry计算机](https://en.wikipedia.org/wiki/Atanasoff%E2%80%93Berry_Computer)”）。它有60个50位字的存储器，以[电容器](https://en.wikipedia.org/wiki/Capacitor)形式（带有刷新电路——第一个再生存储器）安装在两个转鼓上。时钟速度为60Hz，加法需要1秒。对于辅助存储器，它使用穿孔卡片，由用户移动。这些孔实际上并没有在卡片上打孔，而是被烧毁了。打孔卡系统的错误率从未降低到0.001%以上，这是不够的。美国参战后，阿塔纳索夫离开了爱荷华州，结束了他在数字计算机方面的工作。
- 在德国，[HelmutHölzer](https://en.wikipedia.org/wiki/Helmut_H%C3%B6lzer)建造了一台[模拟计算机](https://en.wikipedia.org/wiki/Analog_computer)来计算和模拟 [V-2火箭](https://en.wikipedia.org/wiki/V-2_rocket)弹道。
- 在德国，[KonradZuse](https://en.wikipedia.org/wiki/Konrad_Zuse)开发了S1，这是世界上第一台过程计算机，[Henschel](https://en.wikipedia.org/wiki/Henschel)使用它来测量机翼表面。
# 1943年
- ## 4月
	- 在英国，[MaxNewman](https://en.wikipedia.org/wiki/Max_Newman)、[CEWynn-Williams](https://en.wikipedia.org/wiki/C._E._Wynn-Williams)和他们在英国[布莱切利布莱切利公园](https://en.wikipedia.org/wiki/Bletchley_Park)秘密政府密码和密码学校（“X站”）的团队完成了“[希思·罗宾逊](https://en.wikipedia.org/wiki/Heath_Robinson_(codebreaking_machine))”。这是用于破译密码的专用计数机，不是通用计算器或计算机，而是使用电子和继电器逻辑组合的逻辑设备。它以每秒2000个字符的速度从两个闭合的纸带环中以光学方式读取数据。它很重要，因为它是Colossus的先驱。纽曼在[剑桥大学](https://en.wikipedia.org/wiki/University_of_Cambridge)认识图灵（图灵是纽曼的学生），并且是第一个看到图灵1936年论文草稿的人。 [希思·罗宾逊](https://en.wikipedia.org/wiki/Heath_Robinson)是一位以画滑稽机器而闻名的英国[漫画家](https://en.wikipedia.org/wiki/Cartoonist)的名字，比如美国的[鲁布·戈德堡](https://en.wikipedia.org/wiki/Rube_Goldberg)。该系列中后来的两台机器以伦敦商店的名字命名，名字中带有“Robinson”。
- ## 9月
	- 在美国，Williams和Stibitz完成了“继电器内插器”，后来称为“[模型II继电器计算器](https://en.wikipedia.org/wiki/Bell_Labs#Calculators)”。这是一个可编程计算器；同样，程序和数据是从纸带中读取的。一个创新功能是，为了提高可靠性（错误检测/自检），数字以[双五进制](https://en.wikipedia.org/wiki/Bi-quinary_coded_decimal)格式表示，每个数字使用七个继电器，其中两个应该“打开”：0100001代表0,0100010为1，依此类推，直到1010000为9。本系列中的一些后期机器将使用二元表示法表示浮点数的数字。
- ## 12月
	- 在英国，马克1[巨像](https://en.wikipedia.org/wiki/Colossus_computer)完成，由[汤米·弗劳尔斯](https://en.wikipedia.org/wiki/Tommy_Flowers)在[邮局研究实验室](https://en.wikipedia.org/wiki/Post_Office_Research_Station)在伦敦，以协助在德国的开裂[洛伦兹SZ42密码](https://en.wikipedia.org/wiki/Lorenz_cipher)在布莱切利公园。它是一个二进制数字机器，包含1500个[真空管（阀门）](https://en.wikipedia.org/wiki/Vacuum_tube)，并将可编程逻辑功能应用于字符流，以每秒5000个字符的速度从[穿孔纸带](https://en.wikipedia.org/wiki/Punched_tape)循环中读取和重新读取。它有501[位](https://en.wikipedia.org/wiki/Bit)内存，程序设置在开关和插头面板上。Colossus在二战期间在[BletchleyPark](https://en.wikipedia.org/wiki/Bletchley_Park)使用-作为生产力较低的HeathRobinson机器的后续产品。
# 1944年
- ## 6月
	- 在英国，第一个马克2巨像被委托。它是Mark1机器的开发，包含2400个真空管。它有五个相同的并行处理器，由一个[移位寄存器提供](https://en.wikipedia.org/wiki/Shift_register)，每秒可处理25,000个字符。Colossus可以评估各种[布尔代数](https://en.wikipedia.org/wiki/Boolean_algebra)函数，以帮助建立LorenzSZ42机器的转子设置。到1945年5月欧洲战争结束时，布莱切利公园使用了10辆Mark2Colossi。然后，除了两台机器外，其他所有机器都被拆成小部件，无法推断它们的用途，以保持工作的保密性。其余两个在1960年代在[GCHQCheltenham](https://en.wikipedia.org/wiki/GCHQ)拆除。
- ## 8月
	- ### 7日
		- 在美国，在[IBM](https://en.wikipedia.org/wiki/IBM)自动序列控制计算器被移交给[哈佛大学](https://en.wikipedia.org/wiki/Harvard_University)，调用它的[马克一号](https://en.wikipedia.org/wiki/Harvard_Mark_I)。它由[HowardAiken](https://en.wikipedia.org/wiki/Howard_Aiken)和他的团队设计，由IBM资助和建造——它成为第二台程控机器（在[KonradZuse](https://en.wikipedia.org/wiki/Konrad_Zuse)之后）。整台机器长51英尺（16m），重5（短）吨（4.5吨），包含750,000个零件。它使用3304个机电继电器作为通断开关，有72个累加器（每个都有自己的算术单元），以及一个容量为23位加号的机械寄存器。算术是定点和十进制的，带有[控制面板](https://en.wikipedia.org/wiki/Control_panel_(computer))设置确定小数位数。输入输出设备包括读卡器、打卡机、纸带阅读器和打字机。有60组旋转开关，每组都可以用作常量寄存器——一种机械[只读存储器](https://en.wikipedia.org/wiki/Read-only_memory)。该程序是从一张纸带中读取的；数据可以从其他磁带、读卡器或常量寄存器中读取。条件跳转不可用。然而，在后来的几年里，机器被修改为支持程序的多个纸带阅读器，从一个到另一个的转移是有条件的，就像一个有条件的子程序调用。另一个附加功能允许提供可从磁带调用的插板有线子程序。用于为美国海军创建弹道表.。
# 1945年

- 在美国，[VannevarBush](https://en.wikipedia.org/wiki/Vannevar_Bush)开发了[memex](https://en.wikipedia.org/wiki/Memex)理论，这是一种链接到图书和电影库的[超文本](https://en.wikipedia.org/wiki/Hypertext)设备。
- 在美国，[约翰·冯·诺依曼](https://en.wikipedia.org/wiki/John_von_Neumann)起草了一份报告，描述了最终构建为[EDVAC](https://en.wikipedia.org/wiki/EDVAC)（电子离散变量自动计算机）的未来计算机。[EDVAC报告的初稿](https://en.wikipedia.org/wiki/First_Draft_of_a_Report_on_the_EDVAC)包括对[存储程序计算机](https://en.wikipedia.org/wiki/Stored-program_computer)设计的首次公开描述，从而产生了[冯诺依曼体系结构](https://en.wikipedia.org/wiki/Von_Neumann_architecture)这一术语。它直接或间接地影响了几乎所有后续项目，尤其是[EDSAC](https://en.wikipedia.org/wiki/EDSAC)。设计团队包括[JohnW.Mauchly](https://en.wikipedia.org/wiki/John_W._Mauchly)和[J.PresperEckert](https://en.wikipedia.org/wiki/J._Presper_Eckert)。

# 1948年

# 1949年
- 在澳大利亚，CSIRMkI（后称为[CSIRAC](https://en.wikipedia.org/wiki/CSIRAC)）是澳大利亚的第一台计算机，运行了它的第一个测试程序。它是一种基于真空管的电子通用计算机。它的主存储器将数据作为一系列声波脉冲存储在5英尺（1.5米）长的充满水银的管子中。
- 在英国，[MONIAC](https://en.wikipedia.org/wiki/MONIAC)（货币国民收入模拟计算机）也称为菲利普斯液压计算机，创建于1949年，用于模拟英国的国民经济过程。MONIAC由一系列透明塑料罐和管道组成。人们认为制造了十二到十四台机器。
- ## 3月
	- 在美国，[JohnPresperEckert](https://en.wikipedia.org/wiki/John_Presper_Eckert)和[JohnWilliamMauchly](https://en.wikipedia.org/wiki/John_William_Mauchly)为Northrop构建了[BINAC](https://en.wikipedia.org/wiki/BINAC)。
- ## 5月
	- ### 6日
		- 在英国，这被认为是现代计算的诞生日。 [MauriceWilkes](https://en.wikipedia.org/wiki/Maurice_Wilkes)和[剑桥大学的](https://en.wikipedia.org/wiki/University_of_Cambridge)一个团队在[EDSAC](https://en.wikipedia.org/wiki/EDSAC)计算机上执行了第一个存储程序，该程序使用纸带输入-输出。基于[约翰·冯·诺依曼](https://en.wikipedia.org/wiki/John_von_Neumann)关于存储程序计算机的想法，EDSAC是第一台完整的、功能齐全的冯·诺依曼架构计算机。
- ## 10月
	- 在英国，在[曼彻斯特马克1种](https://en.wikipedia.org/wiki/Manchester_Mark_1)最终规格完成;值得注意的是，这台机器是第一台使用等效基址/[索引寄存器的](https://en.wikipedia.org/wiki/Index_register)计算机，该功能直到1955年左右的第二代才进入通用计算机体系结构。
# 1950年
- 在英国，[图灵测试](https://en.wikipedia.org/wiki/Turing_Test)——英国数学家和计算机先驱[艾伦·图灵](https://en.wikipedia.org/wiki/Alan_Turing)发表了一篇论文，描述了人类和计算机智能和通信的潜在发展。这篇论文后来被称为图灵测试。
- 在美国，哈佛时代杂志封面故事“[马克三世](https://en.wikipedia.org/wiki/Harvard_Mark_III)：人类能造出超人吗？”包括[HowardAiken](https://en.wikipedia.org/wiki/Howard_Aiken)的一句话，评论当时正在建设中的“计算器”（计算机）：“如果我们想让它们忙起来，我们就必须想出更大的问题。”。
- ## 4月
	- 在美国，[国家民族事务委员会](https://en.wikipedia.org/wiki/SEAC_(computer))（东部标准自动计算机）在证明[美国NBS](https://en.wikipedia.org/wiki/National_Institute_of_Standards_and_Technology)在华盛顿特区-是第一个全功能的[存储程序计算机](https://en.wikipedia.org/wiki/Stored-program_computer)在美国。
- ## 5月
	- 在英国，1950年5月10日，带有800个真空管和用于主存储器的水银延迟线的[PilotACE](https://en.wikipedia.org/wiki/Pilot_ACE)计算机在伦敦附近的[国家物理实验室](https://en.wikipedia.org/wiki/National_Physical_Laboratory,_UK)开始运行。它是完整[ACE](https://en.wikipedia.org/wiki/ACE_(computer))的初步版本，由[AlanTuring](https://en.wikipedia.org/wiki/Alan_Turing)设计。
- ## 8月
	- 在美国，[SWAC](https://en.wikipedia.org/wiki/SWAC_(computer))（StandardsWesternAutomaticComputer）在[洛杉矶加州大学](https://en.wikipedia.org/wiki/UCLA)洛杉矶分校展示；在IAS机器之前世界上最快的计算机。
- ## 9月
	- 在德国，[KonradZuse](https://en.wikipedia.org/wiki/Konrad_Zuse)将他的[Z4](https://en.wikipedia.org/wiki/Z4_(computer))机器租给了[苏黎世联邦理工学院](https://en.wikipedia.org/wiki/ETH_Zurich)五年。Z4是一个基于继电器的机器。1949年秋天签订了相应的合同，1950年7月机器运抵苏黎世后在苏黎世重新组装。Z4被[ERMETH](https://en.wikipedia.org/wiki/ERMETH)取代，ERMETH是1953年至1956年在瑞士ETH开发的计算机，是欧洲大陆最早的电子计算机之一。
# 1951年
- 在美国，[EDVAC](https://en.wikipedia.org/wiki/EDVAC)（电子离散变量计算机）。第一台使用磁带的计算机。EDVAC可以从磁带中加载新程序。由约翰·冯·诺依曼(JohnvonNeumann)提出，安装在美国普林斯顿高级研究所。
- 在澳大利亚，[CSIRAC](https://en.wikipedia.org/wiki/CSIRAC)用于播放音乐——这是第一次将计算机用作乐器。
- 在美国，的[A-0](https://en.wikipedia.org/wiki/A-0_System)的高级别编译器是由发明[格雷斯默里料斗](https://en.wikipedia.org/wiki/Grace_Hopper)。
- ## 3月
	- ### 30日
		- 在美国，第一台商业上成功的电子计算机[UNIVAC](https://en.wikipedia.org/wiki/UNIVAC_I)也是第一台通用计算机——旨在处理数字和文本信息。由[J.PresperEckert](https://en.wikipedia.org/wiki/J._Presper_Eckert)和[JohnMauchly设计](https://en.wikipedia.org/wiki/John_Mauchly)，其公司随后转交给了[RemingtonRand](https://en.wikipedia.org/wiki/Remington_Rand)。这台机器的实施标志着计算机时代的真正开始。RemingtonRand向美国人口普查局交付了第一台UNIVAC机器。这台机器使用磁带作为输入。
- ## 4月
	- ### 21日
		- 在美国，[Whirlwind](https://en.wikipedia.org/wiki/MIT_Whirlwind)是由JayForrester团队在[麻省理工学院](https://en.wikipedia.org/wiki/MIT)为美国防空系统制造的第一台实时计算机，现已投入使用。这台计算机是第一个允许[交互计算的计算机](https://en.wikipedia.org/wiki/Interactive_computing)，允许用户使用键盘和阴极射线管与其交互。Whirlwind的设计后来发展成[SAGE](https://en.wikipedia.org/wiki/Semi_Automatic_Ground_Environment)，这是一个综合性的实时计算机系统，用于对空袭进行预警。
- ## 9月
	- 在英国，已知最古老的[计算机生成音乐](https://en.wikipedia.org/wiki/Computer_music)录音是由[FerrantiMark1](https://en.wikipedia.org/wiki/Ferranti_Mark_1)计算机播放的。Mark1是[曼彻斯特](https://en.wikipedia.org/wiki/Manchester_Mark_1)[大学](https://en.wikipedia.org/wiki/Victoria_University_of_Manchester)曼彻斯特Mark1机器的商业版本。音乐节目由[ChristopherStrachey](https://en.wikipedia.org/wiki/Christopher_Strachey)编写。
- ## 11月
	- ### 17日
		- 在英国，JLyons是一家以茶闻名的英国食品公司，它通过在电子计算机上运行第一个商业应用程序创造了历史。工资系统在[里昂电子办公室](https://en.wikipedia.org/wiki/LEO_I)(LEO)上运行，这是一个由[MauriceWilkes](https://en.wikipedia.org/wiki/Maurice_Wilkes)设计的计算机系统，他曾在[EDSAC](https://en.wikipedia.org/wiki/EDSAC)工作过。
# 1952年
- 在苏联，[BESM-1](https://en.wikipedia.org/wiki/BESM)完成。只制造了一台BESM-1机器。该机器使用了大约5,000个真空管。
- ## 4月
	- 在美国，[IBM](https://en.wikipedia.org/wiki/IBM)推出了[IBM701](https://en.wikipedia.org/wiki/IBM_701)，这是其[700和7000系列](https://en.wikipedia.org/wiki/IBM_700/7000_series)大型机器中的第一台计算机，具有各种科学和商业架构，但具有通用的电子设备和外围设备。该系列中的一些计算机一直使用到1980年代。
- ## 6月

# 1953年
- 在英国，在[曼彻斯特大学的](https://en.wikipedia.org/wiki/University_of_Manchester)团队完成了[第一个晶体管化计算机](https://en.wikipedia.org/wiki/Transistor_computer#First_transistor_computer)。
- 在美国，[ArthurAndersen](https://en.wikipedia.org/wiki/Arthur_Andersen)受雇为[通用电气](https://en.wikipedia.org/wiki/General_Electric)([GE](https://en.wikipedia.org/wiki/GE))位于肯塔基州路易斯维尔附近的家电园区制造工厂编制工资单。结果，GE购买了[UNIVACI](https://en.wikipedia.org/wiki/UNIVAC_I)，成为美国第一台商用计算机。JoeGlickauf是[ArthurAndersen](https://en.wikipedia.org/wiki/Arthur_Andersen)参与GE的项目负责人。
- 在世界，估计世界上有100台计算机。
- 在美国，开发磁芯存储器。
# 1954年
- 在美国，[FORTRAN](https://en.wikipedia.org/wiki/FORTRAN)(FORmulaTRANslation)是第一个高级编程语言开发，由[JohnBackus](https://en.wikipedia.org/wiki/John_Backus)和他在IBM的团队开始。开发一直持续到1957年。它仍然用于科学编程。FORTRAN程序在运行前需要通过[编译器](https://en.wikipedia.org/wiki/Compiler)转换成机器程序，本身就是一个程序。
- 在美国，在[IBM650](https://en.wikipedia.org/wiki/IBM_650)被引入。一个相对便宜的带鼓存储的十进制机器，它成为第一台批量生产的计算机，安装了大约2000台。

# 1956年
- 在美国，首次[人工智能](https://en.wikipedia.org/wiki/Artificial_intelligence)会议在新罕布什尔州达特茅斯学院举行。
- 在美国，在[本迪克斯G-15](https://en.wikipedia.org/wiki/Bendix_G-15)计算机通过引入[Bendix公司](https://en.wikipedia.org/wiki/Bendix_Corporation)。
- 在下，[EdsgerDijkstra](https://en.wikipedia.org/wiki/Edsger_Dijkstra)发明了一种用于图中最短路径的有效算法，以证明[ARMAC](https://en.wikipedia.org/w/index.php?titleARMAC&actionedit&redlink1)计算机的能力。使用的例子是荷兰铁路系统。之所以选择这个问题，是因为它可以很快得到解释，并且可以检查结果。尽管这是许多人会记住Dijkstra的主要事情，但他也为许多计算领域做出了重要贡献——尤其是他在与并发相关的问题上所做的工作，例如[信号量](https://en.wikipedia.org/wiki/Semaphore_(programming))的发明，应该会被人们记住。

# 1958年
- 在美国，编程语言[LISP](https://en.wikipedia.org/wiki/Lisp_(programming_language))（解释型）开发，于1960年完成。LISP代表“列表处理”。用于人工智能开发。通过开发[约翰·麦卡锡](https://en.wikipedia.org/wiki/John_McCarthy_(computer_scientist))在[麻省理工学院](https://en.wikipedia.org/wiki/Massachusetts_Institute_of_Technology)。
- 在苏联，[Setun](https://en.wikipedia.org/wiki/Setun)，一种[平衡的三元](https://en.wikipedia.org/wiki/Balanced_ternary)计算机，于1958年在[莫斯科国立大学开发](https://en.wikipedia.org/wiki/Moscow_State_University)。
- ## 9月
	- ### 12日
		- 在美国，该[集成电路](https://en.wikipedia.org/wiki/Integrated_circuit)发明了[杰克·基尔比](https://en.wikipedia.org/wiki/Jack_Kilby)在[德州仪器](https://en.wikipedia.org/wiki/Texas_Instruments)。后来创立[英特尔的](https://en.wikipedia.org/wiki/Intel)罗伯特·诺伊斯(RobertNoyce)也单独致力于这项发明。英特尔后来继续完善微处理器。该专利于1959年申请并于1964年获得批准。该专利未被日本接受，因此日本企业可以避免支付任何费用，但在1989年-经过30年的法律斗争-日本批准了该专利；所以直到2001年之前，所有日本公司都支付了费用——在专利在世界其他地方过时很久之后。
# 1959年
- 在世界，1959年至1964年间推出的计算机，通常被视为第二代计算机，基于分立晶体管和印刷电路——从而产生更小、更强大和更可靠的计算机。
- 在英国，[ChristopherStrachey](https://en.wikipedia.org/wiki/Christopher_Strachey)成为牛津大学第一位计算教授，他提交了[分时](https://en.wikipedia.org/wiki/Time-sharing)专利申请。他在当年在巴黎举行的联合国教科文组织赞助的信息处理会议上将这个概念传递给了[JCRLicklider](https://en.wikipedia.org/wiki/J._C._R._Licklider)。
- 在美国，[COBOL](https://en.wikipedia.org/wiki/COBOL)（通用面向商业语言）由GraceMurrayHopper作为[FLOW-MATIC](https://en.wikipedia.org/wiki/FLOW-MATIC)的继承者开发，于1961年完成。
- 在苏联，[明斯克](https://en.wikipedia.org/wiki/Minsk_family_of_computers) [大型计算机的](https://en.wikipedia.org/wiki/Mainframe_computer)开发和生产始于苏联。1975年停产。
- 在美国，的[MOSFET](https://en.wikipedia.org/wiki/MOSFET)（MOS晶体管）发明通过[穆罕默德Atalla](https://en.wikipedia.org/wiki/Mohamed_Atalla)和[达沃·卡](https://en.wikipedia.org/wiki/Dawon_Kahng)在[贝尔实验室](https://en.wikipedia.org/wiki/Bell_Labs)。在[MOSFET](https://en.wikipedia.org/wiki/MOSFET)（金属-氧化物-半导体[场效应晶体管](https://en.wikipedia.org/wiki/Field-effect_transistor)），也被称为MOS晶体管，被发明[穆罕默德Atalla](https://en.wikipedia.org/wiki/Mohamed_Atalla)和[达沃·卡](https://en.wikipedia.org/wiki/Dawon_Kahng)在[贝尔实验室](https://en.wikipedia.org/wiki/Bell_Labs)。它使[高密度集成电路成为可能](https://en.wikipedia.org/wiki/Very_large-scale_integration)，是计算机中使用最广泛的半导体器件。
- 在美国，的[硅](https://en.wikipedia.org/wiki/Silicon) [集成电路](https://en.wikipedia.org/wiki/Integrated_circuit)由开发[罗伯特·诺伊斯](https://en.wikipedia.org/wiki/Robert_Noyce)在[仙童半导体公司](https://en.wikipedia.org/wiki/Fairchild_Semiconductor)，使用[吉恩·霍尼](https://en.wikipedia.org/wiki/Jean_Hoerni)的[平面工艺](https://en.wikipedia.org/wiki/Planar_process)。与Kilby的[锗](https://en.wikipedia.org/wiki/Germanium) [集成电路不同](https://en.wikipedia.org/wiki/Integrated_circuit)，Noyce使用JeanHoerni的平面工艺开发了一种[硅](https://en.wikipedia.org/wiki/Silicon)集成电路。
- 在美国，贝尔实验室的DouglasE.Eastwood和[DouglasMcIlroy](https://en.wikipedia.org/wiki/Douglas_McIlroy)创建了Macro[SAP](https://en.wikipedia.org/wiki/Symbolic_Assembly_Program)，这是第一种具有高级[宏](https://en.wikipedia.org/wiki/Macro_(computer_science))功能的编程语言。次年，麦克罗伊发表了[宏处理器](https://en.wikipedia.org/wiki/General-purpose_macro_processor)和编程语言可扩展性领域的开创性论文。
# 1960年

- 在英国，第一个[编译器-编译器](https://en.wikipedia.org/wiki/Compiler-compiler)发布。
# 1961年

# 1962年
- 在英国，[ATLAS](https://en.wikipedia.org/wiki/Atlas_Computer)由曼彻斯特大学团队完成。这台机器引入了许多现代架构概念：假脱机、中断、流水线、交错内存、虚拟内存和分页。它在发布时是世界上最强大的机器。
- 在美国，1961年5月，麻省理工学院物理学家韦斯利·A·克拉克(WesleyA.Clark)的心血结晶[LINC](https://en.wikipedia.org/wiki/LINC)开始工作。它是第一个按比例缩小以针对个人用户进行优化和定价的计算机功能原型（约43,600美元–相当于2020年的373,000美元）.1963年在马里兰州贝塞斯达的美国国立心理健康研究所首次使用，许多人认为它是第一台个人计算机，尽管某些元素（例如内存架）的尺寸很大。
- 在美国，[太空战！](https://en.wikipedia.org/wiki/Spacewar!)，一个早期且极具影响力的电脑游戏，由[麻省理工学院](https://en.wikipedia.org/wiki/MIT)学生[史蒂夫罗素](https://en.wikipedia.org/wiki/Steve_Russell_(computer_scientist))编写。游戏在[DEC](https://en.wikipedia.org/wiki/Digital_Equipment_Corporation) [PDP-1上运行](https://en.wikipedia.org/wiki/PDP-1)。竞争玩家使用早期版本的[操纵杆](https://en.wikipedia.org/wiki/Joystick)向彼此的太空飞船开火。
# 1963年
- 在美国，[DouglasEngelbart](https://en.wikipedia.org/wiki/Douglas_Engelbart)设计的鼠标。鼠标直到1983年在[AppleComputer](https://en.wikipedia.org/wiki/Apple_Computer)的[Lisa](https://en.wikipedia.org/wiki/Apple_Lisa)和[Macintosh中](https://en.wikipedia.org/wiki/Apple_Macintosh)才流行起来，直到1987年才被IBM采用——尽管在此日期之前，[Amstrad](https://en.wikipedia.org/wiki/Amstrad) [PC1512](https://en.wikipedia.org/wiki/PC1512)等兼容计算机配备了鼠标。
# 1964年
- 在美国，[PaulBaran](https://en.wikipedia.org/wiki/Paul_Baran)发明了后来被称为[分组交换的东西](https://en.wikipedia.org/wiki/Packet_switching)。Baran在1960年至1964年间发表了一系列关于将信息划分为“消息块”并通过分布式网络发送的简报和论文。
- 在美国，1964年至1972年间制造的计算机通常被视为第三代计算机；它们基于第一个集成电路——创造出更小的机器。此类机器的典型代表是[HP2116A](https://en.wikipedia.org/wiki/HP_2116A)和[DataGeneralNova](https://en.wikipedia.org/wiki/Data_General_Nova)。

- 在美国，推出IBMSystem/360–第一个兼容计算机系列，逆转并阻止了分离的“商业”和“科学”机器架构的演变；所有模型都使用相同的基本指令集架构和寄存器大小，理论上允许程序根据需求变化迁移到或多或少强大的模型。内存的基本单位“字节”被定义为8位，而更大的单位，例如“字”，其大小定义为8的倍数，这会产生许多后果。当时许多竞争计算机使用的字长是6的倍数。此时经常使用营销术语“IBM兼容”来表示该体系结构使用[8位](https://en.wikipedia.org/wiki/8-bit)字节。到1968年，出货量超过14,000辆。

# 1975年
- 在意大利，[Olivetti](https://en.wikipedia.org/wiki/Olivetti)发布了个人电脑[OlivettiP6060](https://en.wikipedia.org/wiki/Olivetti_P6060)，这是第一款带有集成[软盘驱动](https://en.wikipedia.org/wiki/Floppy_disk)器的个人电脑。它是由[皮埃尔·乔治·佩罗托](https://en.wikipedia.org/wiki/Pier_Giorgio_Perotto)(PierGiorgioPerotto)设计的。
- 在美国，在[MITS](https://en.wikipedia.org/wiki/Micro_Instrumentation_and_Telemetry_Systems) [牛郎星8800](https://en.wikipedia.org/wiki/Altair_8800)，第一个商业上成功的爱好计算机，被释放。《[大众电子》](https://en.wikipedia.org/wiki/Popular_Electronics)（1975年1月）上的一篇文章描述了计算机并邀请人们订购套件。尽管处理能力、输入/输出系统（[闪光灯](https://en.wikipedia.org/wiki/Blinkenlights)和拨动开关）和内存（256字节）有限，但第一天就订购了大约200个。10,000件最终以每件397美元的套件价格发货。许多公司生产了基于“S-100总线”（Altair的主总线）的克隆版本。
- 在美国，第一台微型计算机执行的[BASIC](https://en.wikipedia.org/wiki/BASIC)由[比尔·盖茨](https://en.wikipedia.org/wiki/Bill_Gates)和[保罗·艾伦](https://en.wikipedia.org/wiki/Paul_Allen)。它是为MITSAltair编写的。这导致了[微软](https://en.wikipedia.org/wiki/Microsoft)在今年晚些时候的成立。
- 在美国，[Unix](https://en.wikipedia.org/wiki/Unix)上市（见1969）。
- 在挪威，[挪威](https://en.wikipedia.org/wiki/Norway)公司[Mycron](https://en.wikipedia.org/wiki/Mycron)发布了其第一台单板计算机[MYCRO-1](https://en.wikipedia.org/wiki/MYCRO-1)。
- 在美国，形成[微软](https://en.wikipedia.org/wiki/Microsoft)的[比尔·盖茨](https://en.wikipedia.org/wiki/Bill_Gates)和[保罗·艾伦](https://en.wikipedia.org/wiki/Paul_Allen)。
- 在美国，[MOSTechnology,Inc.](https://en.wikipedia.org/wiki/MOS_Technology)发布了他们的[6501](https://en.wikipedia.org/wiki/MOS_Technology_6501)CPU。它与摩托罗拉的6800[引脚兼容](https://en.wikipedia.org/wiki/Pin-compatible)，[摩托罗拉](https://en.wikipedia.org/wiki/Motorola_6800)很快就对它们提起诉讼。在6501迅速停止销售，并与更换[6502](https://en.wikipedia.org/wiki/MOS_Technology_6502)具有“官司兼容”的设计，但在其他方面几乎等同于6501.6502成为最热门的CPU之一，在未来10年，用于许多计算机和游戏机（最著名的是[Atari2600](https://en.wikipedia.org/wiki/Atari_2600)、[AppleII](https://en.wikipedia.org/wiki/Apple_II_family)、[CommodorePET](https://en.wikipedia.org/wiki/Commodore_PET)、[VIC-20](https://en.wikipedia.org/wiki/Commodore_VIC-20)和[Commodore64](https://en.wikipedia.org/wiki/Commodore_64)、[AcornElectron](https://en.wikipedia.org/wiki/Acorn_Electron)/[BBCMicrocomputer](https://en.wikipedia.org/wiki/BBC_Microcomputer)，以及[任天堂娱乐系统](https://en.wikipedia.org/wiki/Nintendo_Entertainment_System)）。
- 在美国，[IBM5100](https://en.wikipedia.org/wiki/IBM_5100)电脑发布；它集成了键盘、显示器和磁带上的大容量存储设备，虽然不使用微处理器，但它类似于几年后的[个人计算机](https://en.wikipedia.org/wiki/Personal_computer)。
- 在意大利，[CSELT](https://en.wikipedia.org/wiki/CSELT)实验室发布了[MUSA（MultichannelspeakAutomaton）](https://en.wikipedia.org/wiki/MUSA_(MUltichannel_Speaking_Automaton))，[语音合成](https://en.wikipedia.org/wiki/Speech_Synthesis)的早期实验。它能够阅读和演唱带有多种声音的意大利语（[无伴奏合唱](https://en.wikipedia.org/wiki/A_cappella)）。
- ## 11月
	- 在美国，[Zilog](https://en.wikipedia.org/wiki/Zilog)由前英特尔员工创立。
# 1976年
- 在美国，IBM推出的第一台[激光打印机](https://en.wikipedia.org/wiki/Laser_printer)——IBM3800。第一个彩色版本于1988年上市。
- 在美国，在介绍[英特尔8085](https://en.wikipedia.org/wiki/Intel_8085)芯片。8080的改进版，与8080S指令集（只有一对夫妇的一个超集的额外的指令）。单5V电源（而8080需要几种不同的电压）。
- 在美国，Zilog发布的Z80芯片。它是8080芯片的超集，具有额外的寄存器和指令，并且仅使用一个电源电压。CP/M最初是为8080编写的，但许多实现使用了Z80。Z80是家用计算机的处理器，例如1977年的[Tandy](https://en.wikipedia.org/wiki/Tandy_computer) [TRS-](https://en.wikipedia.org/wiki/TRS-80)80、1982年的Sinclair[ZXSpectrum](https://en.wikipedia.org/wiki/ZX_Spectrum)等等。
- 在美国，[MOSTechnology,Inc](https://en.wikipedia.org/wiki/MOS_Technology)推出[KIM-1](https://en.wikipedia.org/wiki/KIM-1)微型计算机系统作为其[6502](https://en.wikipedia.org/wiki/MOS_Technology_6502)CPU的演示器。
- 在美国，[Cray-1](https://en.wikipedia.org/wiki/Cray-1)超级计算机是由[SeymourCray](https://en.wikipedia.org/wiki/Seymour_Cray)发明的。他于1972年离开[ControlData](https://en.wikipedia.org/wiki/Control_Data)成立了自己的公司。这台机器以其马蹄形设计而闻名，因为它是第一台使[矢量处理](https://en.wikipedia.org/wiki/Vector_processing)实用的超级计算机。85架以每架500万美元的价格运送。
- 在美国，[Commodore](https://en.wikipedia.org/wiki/Commodore_International)在股票交易中收购了[MOSTechnology,Inc。](https://en.wikipedia.org/wiki/MOS_Technology)MOS价值1200万美元。[ChuckPeddle](https://en.wikipedia.org/wiki/Chuck_Peddle)加入Commodore担任总工程师。购买MOS后，Commodore开始着手开发[CommodorePET](https://en.wikipedia.org/wiki/Commodore_PET)。
- 在美国，创建了Emacs文本编辑软件。
- "[5.25英寸软盘问世](https://en.wikipedia.org/wiki/5.25_inch_floppy_disks)。当该产品进入PC市场时，它会引起数字信息存储的爆炸式增长。"。
- ## 4月
	- ### 1日
		- 在美国，[AppleComputer](https://en.wikipedia.org/wiki/Apple_Computer),Inc.成立，以销售由史蒂夫·沃兹尼亚克(SteveWozniak)和史蒂夫·乔布斯(SteveJobs)设计的[AppleI](https://en.wikipedia.org/wiki/Apple_I)单板计算机。它使用MOSTechnology6502微处理器。
# 1977年
- 在美国，[Commodore](https://en.wikipedia.org/wiki/Commodore_International)介绍了[CommodorePET](https://en.wikipedia.org/wiki/Commodore_PET)。它带有4KB或8KB的RAM，以及一个集成式磁带录音机和9"单色显示器。
- ## 6月
	- ### 5日
		- 在美国，推出的AppleII计算机基于8位[MOS技术](https://en.wikipedia.org/wiki/MOS_Technology) [6502](https://en.wikipedia.org/wiki/MOS_Technology_6502)微处理器，运行频率为1MHz，[RAM](https://en.wikipedia.org/wiki/Random_Access_Memory)为4KB。它具有[开放式架构](https://en.wikipedia.org/wiki/Open_architecture)、使用彩色图形和用于加载程序和存储数据的录音带接口。后来，在1978年7月，软盘驱动器问世，其界面设计精美。1979年，第一个“[杀手级应用](https://en.wikipedia.org/wiki/Killer_app)”（适用于商业世界）的例子之一[——VisiCalc](https://en.wikipedia.org/wiki/VisiCalc)电子表格程序——发布。
- ## 8月
	- 在美国，[Tandy](https://en.wikipedia.org/wiki/Tandy_Corporation)推出了带有“LevelIBASIC”的[TRS-80](https://en.wikipedia.org/wiki/TRS-80)。尽管采用简单的黑白图形，但由于遍布全国的RadioShack连锁店，它迅速成为畅销书。
- ## 9月
	- 在美国，[Heathkit](https://en.wikipedia.org/wiki/Heathkit)提供了[H8](https://en.wikipedia.org/wiki/Heathkit_H-8)家用电脑套件。它基于Intel8080a处理器，附带HDOS（Heathkit磁盘操作系统）和BentonHarborBASIC。
# 1978年
- 在美国，Tandy将TRS-80升级为大大改进的[Microsoft](https://en.wikipedia.org/wiki/Microsoft)8K“LevelIIBASIC”，以及一个“扩展接口”，增加了32KBRAM、一张软盘和一个打印机接口。有了这些附加功能，TRS-80成为了一款可行的小型商务计算机。
- 在给，街机电子游戏《[太空侵略者》](https://en.wikipedia.org/wiki/Space_Invaders)发布，引发电子游戏热潮。1979年，Atari的Asteroids被证明非常受欢迎。
- ## 6月
	- ### 8日
		- 在美国，推出16位[Intel8086](https://en.wikipedia.org/wiki/Intel_8086)，第一个[x86](https://en.wikipedia.org/wiki/X86)微处理器。可用的时钟频率为5、8和10MHz，[指令集](https://en.wikipedia.org/wiki/X86_instruction_listings)约为300操作。在推出时，最快的8086是8MHz版本，它实现了0.8MIPS并包含29,000个晶体管。三十多年后，x86仍然是个人计算历史上最受欢迎和商业上最成功的[指令集架构](https://en.wikipedia.org/wiki/Instruction_set_architecture)。
# 1979年
- 在美国，由JeanIchbiah和霍尼韦尔团队为美国国防部引入的编程语言[Ada](https://en.wikipedia.org/wiki/Ada_(programming_language))。
- 在英国，[CommodorePET](https://en.wikipedia.org/wiki/Commodore_PET)在英国发布。基于1MHz6502处理器，它显示单色文本并且只有8KB的RAM。售价569英镑。16KBRAM的版本售价776英镑，而32KBRAM的售价为914英镑。
- 在NEDJAP，发明了光盘。
- 在美国，摩托罗拉推出的[68000](https://en.wikipedia.org/wiki/68000)微处理器，是68k系列中的第一款。5年后，它被用于AppleMacintosh、AtariST和CommodoreAmiga等机器中。
- 在美国，在V7Unix发布后不久，其中包括[UUCP](https://en.wikipedia.org/wiki/UUCP)（一种通过标准电话线进行通信的协议），TomTruscott和JimEllis开发并发布了[Usenet](https://en.wikipedia.org/wiki/Usenet)，一个全球讨论组系统。如今，它使用[Internet](https://en.wikipedia.org/wiki/Internet)协议，并且仍然很流行。
- 在美国，四名心怀不满的[Atari](https://en.wikipedia.org/wiki/Atari)程序员离开并成立了[动视](https://en.wikipedia.org/wiki/Activision)，这是第一家第三方视频游戏软件发行商。动视同时推广游戏和程序员，改变了软件的营销方式。
- 在美国，TexasInstruments发布[TI-99/4微型计算机](https://en.wikipedia.org/wiki/TI-99/4)。该系统通常使用[录音](https://en.wikipedia.org/wiki/Compact_Cassette)带来存储信息，以及类似于游戏机的ROM模块来保存商业软件。此外，TI还为TI-99/4及其后继产品4A提供了基于他们自己[芯片](https://en.wikipedia.org/wiki/TMS5220)的语音合成器。
- 在美国，[VisiCalc](https://en.wikipedia.org/wiki/VisiCalc)电子表格软件发布。
- 在美国，[WordStar](https://en.wikipedia.org/wiki/WordStar)文字处理软件发布。

	- ### 1日

- ## 11月
	- 在美国，Atari发布了[Atari400/800](https://en.wikipedia.org/wiki/Atari_8-bit_family)，这是一款基于6502微处理器的高性能游戏家用电脑。
# 1980年
- 在荷兰/日本，音频CD上的红皮书是由索尼和飞利浦推出的。这是光盘的开始；它在日本发行，一年后又在欧洲和美国发行。
- ## 1月
	- 在英国，[Sinclair](https://en.wikipedia.org/wiki/Sinclair_Research) [ZX80的](https://en.wikipedia.org/wiki/Sinclair_ZX80)售价低于100英镑。
- ## 5月
	- ### 22日
		- 在日本，游戏[吃豆人](https://en.wikipedia.org/wiki/Pac-Man)发布。
- ## 6月
	- 在美国，Commodore发布了[VIC-20](https://en.wikipedia.org/wiki/Commodore_VIC-20)，它有3.5KB的可用内存，基于[MOSTechnology6502](https://en.wikipedia.org/wiki/MOS_Technology_6502)处理器。杂志变得可用，其中包含各种实用程序和游戏的代码。有一个5¼"[磁盘驱动器](https://en.wikipedia.org/wiki/Floppy_disk)，以及一个使用标准录音带的盒式存储系统。另外还有一些游戏、一个打印在6英寸（152毫米）宽纸带上的彩色绘图仪和一个图形输入板（该[KoalaPad](https://en.wikipedia.org/wiki/KoalaPad)）。电视屏幕担任班长。在VIC-20成为第一个电脑销售100万台。
- ## 7月
	- 在美国，Tandy发布了[TRS-80彩色计算机](https://en.wikipedia.org/wiki/TRS-80_Color_Computer)，它基于摩托罗拉6809E处理器并使用MicrosoftBasic作为其编程语言。它是第一台支持彩色图形的Tandy计算机，还支持卡带程序和游戏，试图在家庭计算和视频游戏市场之间架起桥梁。
- ## 10月
	- 在美国，MS-DOS/PCDOS的开发开始了。微软（主要以其编程语言而闻名）被委托为PC编写操作系统；DigitalResearch没有拿到合同（关于这个的真正原因有很多传说）。DR的操作系统CP/M-86是后来发货的，但实际上使程序适应DOS比适应CP/M-86更容易，而CP/M-86的价格为495美元。由于微软没有操作系统可供出售，他们购买了由TimPaterson编写的西雅图计算机产品公司的[86-DOS](https://en.wikipedia.org/wiki/86-DOS)那年早些时候（86-DOS也被称为QDOS、Quick&Dirty操作系统；它或多或少是CP/M的16位版本）。这些权利实际上是在1981年7月购买的。据说IBM在对操作系统进行审查并重新编写大部分代码时发现了300多个代码中的错误。TimPaterson的DOS1.0是4000行汇编程序。
# 1981年
- 在美国，[理查德·费曼提](https://en.wikipedia.org/wiki/Richard_Feynman)出了[量子计算机](https://en.wikipedia.org/wiki/Quantum_computer)。他想到的主要应用是量子系统的模拟，但他也提到了解决其他问题的可能性。
- 在美国，在[施乐8010（“星”）系统](https://en.wikipedia.org/wiki/Xerox_Star)，第一个商业系统使用WIMP（窗口，图标，菜单和定位设备）的图形用户界面。Apple在[AppleLisa](https://en.wikipedia.org/wiki/Apple_Lisa)界面的开发中融入了其中的许多想法（参见1983年1月）。
- 在美国，[Symbolics](https://en.wikipedia.org/wiki/Symbolics)推出了LM-2工作站，这是一个基于MITCADR架构的基于[Lisp](https://en.wikipedia.org/wiki/Lisp_(programming_language))的工作站。

- ## 8月
	- ### 12日
		- 在美国，IBM宣布了他们的[开放式架构](https://en.wikipedia.org/wiki/Open_architecture) [IBM个人计算机](https://en.wikipedia.org/wiki/IBM_Personal_Computer)。圣诞节前收到了100,000份订单。该设计比IBM预期的要成功得多，并成为大多数现代个人计算机行业的基础。[MDA（单色显示适配器）](https://en.wikipedia.org/wiki/IBM_Monochrome_Display_Adapter)，纯文字，随IBMPC一起引入。[MS-DOS](https://en.wikipedia.org/wiki/MS-DOS)1.0，PCDOS1.0。微软（主要以其编程语言而闻名）受IBM委托编写操作系统；他们从TimPaterson那里购买了一个名为86-DOS的程序，该程序松散地基于CP/M-80。微软的最终程序由IBM销售为PCDOS，微软销售为MS-DOS；后续版本的合作一直持续到1991年的5.0版。与现代版本的DOS相比，版本1非常基础。最显着的区别是每个磁盘上只有一个目录，即根目录。直到2.0版（1983年3月）才支持子目录。[MS-DOS](https://en.wikipedia.org/wiki/MS-DOS)是主要的操作系统，所有IBM-PC兼容电脑，直至微软发布Windows95，根据微软的说法，在1994年，MS-DOS是大约1亿个计算机上运行的全世界。
- ## 9月
	- 在美国，该[TCP/IP](https://en.wikipedia.org/wiki/TCP/IP)协议被建立。这是通过Internet传输大部分信息的协议。
# 1982年
- 在美国，80186/80188的介绍。这些在个人计算机中很少使用，因为它们包含一个内置的DMA和定时器芯片——因此寄存器地址与IBMPC不兼容。
- ## 1月
	- 在英国，在介绍[BBC科技](https://en.wikipedia.org/wiki/BBC_Micro)的基础上，于1981年12月宣布[MOS技术6502](https://en.wikipedia.org/wiki/MOS_Technology_6502)处理器，它是英国学校到发展的一个非常流行的计算机[橡子阿基米德](https://en.wikipedia.org/wiki/Acorn_Archimedes)（1987年）。1984年，政府提出为此类计算机支付一半的费用，以促进其在中等教育中的使用。
	- 在美国，Commodore在拉斯维加斯的消费电子展上推出了Commodore64。C64围绕VIC-II视频集成电路和SID声音接口设备芯片仅用了两个月就建成，使用6510处理器访问64K的RAM和16K的可切换ROM。在接下来的十年中，这款“8位计算机的缩影”销量高达2200万台。
- ## 2月
	- 在美国，2月1日，[80286](https://en.wikipedia.org/wiki/Intel_80286)处理器发布。它实现了新的操作模式，保护模式-允许访问更大的内存（高达16[MB](https://en.wikipedia.org/wiki/Megabyte)相比，1MB为8086）。在推出时，最快的版本运行速度为12.5MHz，达到2.7[MIPS](https://en.wikipedia.org/wiki/Million_instructions_per_second)，包含134,000个晶体管。
- ## 3月
	- 在美国，MS-DOS1.25,PCDOS1.1。
- ## 4月
	- 在英国，Sinclair[ZXSpectrum](https://en.wikipedia.org/wiki/ZX_Spectrum)已于今年晚些时候发布。它基于Zilog的[Z80](https://en.wikipedia.org/wiki/Zilog_Z80)微处理器，运行频率为3.5MHz，带有8色图形显示器。Spectrum出售时带有两种内存选项，16KB版本售价125英镑，48KB版本售价175英镑。

	- 在美国，IBM推出[双面](https://en.wikipedia.org/wiki/Double-sided_disk)320KB软盘驱动器。

	- 在英国/美国，Timex/Sinclair推出了第一台在美国销售的价格低于100美元的计算机，[TimexSinclair1000](https://en.wikipedia.org/wiki/Timex_Sinclair_1000)。尽管早期版本存在缺陷，但仅在前6个月内就售出了50万台，超过了Apple、Tandy和Commodore的总销量。
- ## 8月
