# 公元前4000年
- [Quipu](/wiki/Quipu)\-南美洲安第斯山脉的蒂瓦纳库人的祖先用来计数的打结的绳子。
# 公元前2500年
- 该[算盘](/wiki/Abacus)，第一个已知的计算器，可能是由发明[的巴比伦人](/wiki/Babylonians)作为援助简单的[算术](/wiki/Arithmetic)解决此时间段。它为[位置符号](/wiki/Positional_notation)和后来的[计算](/wiki/Computing)发展奠定了基础。
# 公元前1770年
- 古埃及人在会计文本中首次使用[零](/wiki/0_(number))。
# 公元前910年
- 指南[车](/wiki/South-pointing_chariot)是[中国古代](/wiki/History_of_China#Ancient_China)发明的。它是第一个已知的使用[差速齿轮的](/wiki/Differential_gear)齿轮机构。战车是一辆两轮车，车上有一个指示图形，通过差速器与车轮相连。通过精心选择车轮尺寸、轨道和齿轮比，战车顶部的人物始终指向同一个方向。
# 公元前500年
- [印度语法学家](/wiki/Vyakarana) [Pāṇini](/wiki/P%C4%81%E1%B9%87ini)制定了[梵文语法](/wiki/Sanskrit_grammar)（3959年规则），被称为[Ashtadhyayi](/wiki/Ashtadhyayi)，它是高度系统化和技术性的。Pāṇini使用元[规则](/wiki/Metarule)、[转换](/wiki/Transformation_(mathematics))和[递归](/wiki/Recursion)，其复杂程度以至于他的语法具有相当于[图灵机](/wiki/Turing_machine)的计算能力。Pāṇini的工作是现代[形式语言理论](/wiki/Formal_language_theory)的先驱，也是其在现代计算中使用的先驱。用于描述大多数现代编程语言的[Panini-Backus形式](/wiki/Panini%E2%80%93Backus_form)也与Pāṇini的语法规则非常相似。
# 公元前200年
- [印度数学家](/wiki/Indian_mathematics) [Pingala](/wiki/Pingala)首先描述了[二进制数系统](/wiki/Binary_number_system)，该系统现在基本上用于所有现代计算设备的设计。他还构思了类似于摩尔斯电码的[二进制代码](/wiki/Binary_code)的概念。
# 公元前125年
- 该[安提凯希拉机制](/wiki/Antikythera_mechanism)：发条，[模拟计算机](/wiki/Analog_computer)相信已设计和建造的科林斯殖民地[雪城](/wiki/Syracuse,_Sicily)。该机构包含一个[差动齿轮](/wiki/Differential_gear)，能够跟踪所有当时已知的天体的相对位置。
# 9年
- [中国数学家](/wiki/Chinese_mathematics)首先使用[负数](/wiki/Negative_numbers)。
# 60年
- [亚历山大的英雄](/wiki/Hero_of_Alexandria)做出了许多发明，包括“顺序控制”，其中机器的操作员设置机器运行，然后以确定性的方式遵循一系列指令。这基本上是第一个[程序](/wiki/Computer_program)。他还在自动机领域做出了无数创新，这是[机器人技术](/wiki/Robotics)发展的重要一步。
# 190年
- 在首先提及[算盘](/wiki/Suanpan)（中国珠算），它被广泛使用，直到现代计算器的发明，并继续在一些文化中今天使用。
# 639年
- [印度数学家](/wiki/Indian_mathematics) [Brahmagupta](/wiki/Brahmagupta)是第一个描述现代[位值](/wiki/Place-value) [数字系统](/wiki/Numeral_system)（[印度数字系统](/wiki/Hindu_numeral_system)）的人。
# 725年
- 中国发明家[梁令赞](/wiki/Liang_Lingzan)建造了世界上第一座全机械钟；[水钟](/wiki/Water_clocks)，其中一些非常准确，在此之前已经为人所知了几个世纪。这是一次重要的技术飞跃；一千年后制造的最早的真正计算机使用基于时钟的技术。
# 820年
- [波斯数学家](/wiki/Islamic_mathematics)，[花拉子米](/wiki/Mu%E1%B8%A5ammad_ibn_M%C5%ABs%C4%81_al-Khw%C4%81rizm%C4%AB)，描述现代的雏形[代数](/wiki/Algebra)的名字是从他的书中得到的[铝基塔AL-FImuḫtaṣar人ḥisāb-贾布尔WA-1-muqābala](/wiki/The_Compendious_Book_on_Calculation_by_Completion_and_Balancing)。这个词[算法](/wiki/Algorithm)，距离Al-花拉子米的拉丁名称派生Algoritmi。
# 850年
- [阿拉伯数学家](/wiki/Islamic_mathematics)，[肯迪](/wiki/Al-Kindi)（Alkindus），当时的先驱[密码](/wiki/Cryptography)。他在AManuscriptonDecipheringCryptographicMessages中给出了第一个已知的[密码分析](/wiki/Cryptanalysis)记录解释。特别是，他因开发了频率分析方法而受到赞誉，由此可以分析和利用字母出现频率的变化来破解加密密码（即通过频率分析进行密码分析）。本书还介绍了密码分析、加密、某些加密的密码分析以及阿拉伯语字母和字母组合的统计分析的方法。
- 该[班·马萨](/wiki/Ban%C5%AB_M%C5%ABs%C4%81)兄弟，在他们的[巧妙手法的书](/wiki/Book_of_Ingenious_Devices)，发明了“已知最早的机械[乐器](/wiki/Musical_instrument)”，在这种情况下[hydropowered](/wiki/Hydropower) [器官](/wiki/Organ_(music))，其自动播放互换缸。直到19世纪下半叶，这种“表面带有凸起销钉的圆柱体仍然是机械地制作和再现音乐的基本装置”。他们还发明了一种[自动](/wiki/Automaton) [长笛演奏](/wiki/Flute)器，这似乎是第一台[可编程](/wiki/Computer_programming)机器。
# 1000年
- [AbūRayhānal-Bīrūnī](/wiki/Ab%C5%AB_Rayh%C4%81n_al-B%C4%ABr%C5%ABn%C4%AB)发明了[Planisphere](/wiki/Planisphere)，一种[模拟计算机](/wiki/Analog_computer)。他还发明了第一[机械](/wiki/Mechanical_engineering) [阴阳历](/wiki/Lunisolar_calendar)其中使用的[齿轮系](/wiki/Gear_train)和八个[齿轮](/wiki/Gear)\-wheels。这是一个固定有线知识处理[机器](/wiki/Machine)的早期例子。
# 1015年
- [阿拉伯天文学家](/wiki/Islamic_astronomy)，安达卢斯的[AbūIshāqIbrāhīmal-Zarqālī](/wiki/Ab%C5%AB_Ish%C4%81q_Ibr%C4%81h%C4%ABm_al-Zarq%C4%81l%C4%AB)(Arzachel)发明了赤道仪，这是一种机械模拟计算机设备，用于无需计算即可找到月球、太阳和行星的经度和位置，使用几何图形模型来表示天体的平均和异常位置。
# 1150年
- [阿拉伯天文学家](/wiki/Islamic_astronomy)，[贾比尔·伊本·阿弗拉](/wiki/Jabir_ibn_Aflah)（迦别），发明了[Torquetum](/wiki/Torquetum)，一个观察仪器和机械[模拟计算机](/wiki/Analog_computer)使用之间变换装置[球形坐标系统](/wiki/Spherical_coordinate_system)。它被设计用来获取和转换在三组坐标中进行的测量：[地平线](/wiki/Horizon)、[赤道](/wiki/Equator)和[黄道](/wiki/Ecliptic)。
# 1206年
- [阿拉伯工程师](/wiki/Inventions_in_the_Islamic_world)，[加扎利](/wiki/Al-Jazari)，发明了无数的[自动机](/wiki/Automaton)和提出了许多其他技术创新。其中之一是一个设计[的可编程](/wiki/Program_(machine)) [人形](/wiki/Humanoid_robot)形[时装模特](/wiki/Mannequin)：这似乎是第一次认真，科学的（而不是神奇）的计划[机器人](/wiki/Robot)。他还发明了“[城堡钟](/wiki/Castle_clock)”，一种被认为是最早的可编程模拟计算机的[天文钟](/wiki/Astronomical_clock)。它显示了黄道带、太阳和月球的轨道、一个 [月牙](/wiki/Lunar_phase)形[指针](/wiki/Pointer_(computer_programming))旅游跨越造成自动门打开每小时网关，和五个[机器人](/wiki/Robot)敲击时通过杠杆由操作谁玩音乐的音乐家[凸轮轴](/wiki/Camshaft)连接到[水车](/wiki/Water_wheel)。白天和黑夜的长度可以每天重新编程，以考虑全年白天和黑夜的长度变化。
# 1235年
- [伊斯法罕的](/wiki/Isfahan)波斯天文学家AbiBakr根据AbūRayhānal-Bīrūnī的机械日历模拟计算机的设计，发明了一种带有齿轮日历机芯的黄铜[星盘](/wiki/Astrolabe)。AbiBakr的齿轮星盘使用一组齿轮，是现存最古老的完整机械齿轮机器。
# 1300年
- [RamonLlull](/wiki/Ramon_Llull)发明了LullianCircle：一种通过逻辑组合计算哲学问题（在这种情况下，与基督教有关）答案的概念机器。这个想法在几个世纪后被[莱布尼茨](/wiki/Gottfried_Leibniz)采纳，因此成为计算和[信息科学的基本](/wiki/Information_science)要素之一。
# 1412年
- [艾哈迈德沙·阿尔·坎尔基香迪](/wiki/Ahmad_al-Qalqashandi)给出的列表[密码](/wiki/Cipher)在他Subh阿尔·阿奥沙其中包括[替代](/wiki/Substitution_cipher)和[换位](/wiki/Transposition_cipher)，并在第一时间，用多重替换为每一个密码[明文](/wiki/Plaintext)字母。他还给出了[密码分析的](/wiki/Cryptanalysis)说明和工作示例，包括使用[字母频率](/wiki/Letter_frequencies)表和不能在一个单词中一起出现的字母组。
# 1416年
- [Jamshīdal-Kāshī](/wiki/Jamsh%C4%ABd_al-K%C4%81sh%C4%AB)发明了合相板，这是一种[模拟计算机](/wiki/Analog_computer)仪器，用于确定一天中[行星合相](/wiki/Planetary_conjunction)发生的时间，并用于执行[线性插值](/wiki/Linear_interpolation)。他还发明了一种机械的“行星电脑”，他叫区的板块，这可能图形解决了许多行星的问题，包括真实位置在预测[经度](/wiki/Longitude)太阳和月亮的，和[行星](/wiki/Planet);太阳、月亮和行星的[纬度](/wiki/Latitude)；和[黄道](/wiki/Ecliptic)太阳的。该仪器还集成了[alhidade](/wiki/Alhidade)和[尺子](/wiki/Ruler)。
# 1450年
- [天文学和数学的喀拉拉学校](/wiki/Kerala_school_of_astronomy_and_mathematics)在[南印度](/wiki/South_India)发明了[浮点](/wiki/Floating-point)数字系统。
# 1493年
- [列奥纳多·达·芬奇(LeonardodaVinci)](/wiki/Leonardo_da_Vinci)绘制了一个由互锁齿轮组成的设备图纸，可以将其解释为能够进行加法和减法的[机械计算器](/wiki/Mechanical_calculator)。受该计划启发的工作模型建于1968年，但莱昂纳多是否真的想到了计算器仍然存在争议。达芬奇还为机械人制定了计划：早期的[机器人](/wiki/Robot)设计。
# 1614年
- 苏格兰人[约翰·纳皮尔(JohnNapier)](/wiki/John_Napier)重新发明了一种对数形式和巧妙的可移动杆系统（1617年，被称为纳皮尔的杆或[纳皮尔的骨头](/wiki/Napier%27s_bones)）。这些杆基于[格子或gelosia乘法](/wiki/Lattice_multiplication)算法，允许操作员通过移动杆并将它们放置在特殊构造的板上来乘、除和计算平方根和立方根。
# 1622年
- [WilliamOughtred](/wiki/William_Oughtred)根据JohnNapier开发的对数开发了[计算尺](/wiki/Slide_rule)。
# 1623年
- 德国[博学者](/wiki/Polymath) [威廉·席卡德](/wiki/Wilhelm_Schickard)(WilhelmSchickard)在他寄给约翰内斯·开普勒(JohannesKepler)的两封信上画了一个他称之为计算时钟的装置。一个是在1623年，另一个是在1624年。1624年正在建造的机器后来被一场大火烧毁，他决定放弃他的项目。这台机器直到1957年这两个字母被发现时才为世人所知。一些复制品建于1961年。这台机器对机械计算器的发展没有影响。
# 1642年
- 在法国，法国博学者[布莱斯·帕斯卡](/wiki/Blaise_Pascal)发明了机械计算器。被称为机器算术、帕斯卡的计算器和最终的[帕斯卡林](/wiki/Pascaline)，它在1645年的公开介绍开始了机械计算器的开发，首先是在欧洲，然后是世界其他地方。它是第一台具有受控运载机构的机器。Pascal在发布他的第一台机器之前制造了50台原型机（最终制造了20台机器）。帕斯卡林启发了[GottfriedLeibniz](/wiki/Gottfried_Leibniz)（1671年）、[ThomasdeColmar](/wiki/Thomas_de_Colmar)（1820年）和[DorrE.Felt](/wiki/Dorr_E._Felt)（1887年）的作品。
# 1666年
- 在英国，英国的[塞缪尔·莫兰](/wiki/Samuel_Morland)爵士（SirSamuelMorland，1625-1695年）生产了一种非十进制加法机，适用于[英国货币](/wiki/Pound_sterling)。它没有使用进位机制，而是在辅助拨号上注册进位，用户从中重新输入它们作为加数。
# 1672年
- 在德国，德国[数学家](/wiki/Mathematician)，[莱布尼兹](/wiki/Gottfried_Leibniz)开始设计一种机器，其事半功倍，“[阶梯清算者](/wiki/Stepped_Reckoner)”。它可以将最多5位和12位的数字相乘，得到16位的结果。制造了两台机器，一台在1694年（1879年在阁楼中被发现），另一台在1706年。
# 1685年
- 在德国，戈特弗里德莱布尼茨在一篇题为“在几乎没有分界线的情况下不仅可以执行加减法和乘法的算术机器”的文章中，[戈特弗里德莱布尼茨](/wiki/Gottfried_Leibniz)描述了一种使用[带有活动齿的轮子](/wiki/Pinwheel_calculator)的机器，当它与帕斯卡林耦合时，可以执行所有四个数学运算。没有证据表明莱布尼茨曾经建造过这种风车机器。
# 1709年
- 在意大利，[GiovanniPoleni](/wiki/Giovanni_Poleni)是第一个使用[风车设计](/wiki/Pinwheel_calculator)构建计算器的人。它是用木头做的，做成一个计算钟的形状。
# 1726年
- 在英国，[乔纳森·斯威夫特](/wiki/Jonathan_Swift)（JonathanSwift）在他的《[格列佛游记](/wiki/Gulliver%27s_Travels)》中（讽刺地）描述了一台机器（“引擎”）。“引擎”由一个木制框架和包含词性的木块组成。当发动机的40个杠杆同时转动时，机器显示语法句子片段。
# 1774年
- 在德国，[菲利普·马特乌斯·哈恩](/wiki/Philipp_Matth%C3%A4us_Hahn)(PhilippMatthäusHahn)在现在的德国制造了一款成功的便携式计算器，能够执行所有四种数学运算。
# 1775年
- 在英国，[查尔斯·斯坦霍普(CharlesStanhope)，](/wiki/Charles_Stanhope,_3rd_Earl_Stanhope)英格兰第三代伯爵斯坦霍普(CharlesStanhope)，设计并建造了一个成功的乘法计算器，类似于莱布尼茨(Leibniz)的计算器。
# 1786年
- 在德国，[JHMüller](/wiki/J._H._M%C3%BCller)是Hessian军队的一名工程师，他首先想到了[差分机](/wiki/Difference_engine)的想法（第一次书面提及差分机的基本原理可以追溯到1784年）。
# 1804年
- 在法国，[Joseph-MarieJacquard](/wiki/Joseph-Marie_Jacquard)开发了[Jacquard织机](/wiki/Jacquard_loom)，一种由[穿孔卡片](/wiki/Punched_card)控制的自动织机。
# 1820年
- 在法国，[CharlesXavierThomasdeColmar](/wiki/Charles_Xavier_Thomas_de_Colmar)发明了“[算术计](/wiki/Arithmometer)”，经过三十多年的发展，它于1851年成为第一台批量生产的机械计算器。通过对结果使用可移动累加器，操作员可以快速有效地执行[长乘法](/wiki/Multiplication_algorithm)和除法运算。这台机器基于Pascal和Leibniz的早期作品。
# 1822年
- 在英国，[查尔斯·巴贝奇](/wiki/Charles_Babbage)设计了他的第一台机械计算机，这是用于制表多项式的小数[差分引擎](/wiki/Difference_engine)的第一个原型。
# 1831年
- 在意大利，[GiovanniPlana](/wiki/Giovanni_Plana)设计了一台[万年历机](/wiki/Cappella_dei_Mercanti_(Turin)#Perpetual_calendar)，它可以精确计算超过4000年的日历，计算闰年和昼长变化。
# 1832年
- 在俄罗斯，[SemenKorsakov](/wiki/Semen_Korsakov)提议使用[穿孔卡片](/wiki/Punched_card)来存储和搜索信息。他设计了几台机器来展示他的想法，包括所谓的线性内窥镜。
- 在英国，巴贝奇和[约瑟夫克莱门特](/wiki/Joseph_Clement)制作了他的原型段[差发动机](/wiki/Difference_engine)，，其在6个位数字和二阶差（即，它可以制成表二次多项式）操作。完整的引擎本来是房间大小，计划在大约20位数字的六阶差分和30位数字的三阶差分上运行。每个加法都分两个阶段完成，第二个阶段处理第一个阶段产生的任何进位。输出数字将被冲压到一块软金属板上，用它可能已经制成了印刷板。但是遇到了各种各样的困难，最终完成的只是这件原型作品。
# 1833年
- 在英国，巴贝奇构思并开始设计他的十进制“[分析引擎](/wiki/Analytical_Engine)”。它的[程序](/wiki/Computer_program)以穿孔卡片的形式[存储](/wiki/Computer_data_storage)在[只读存储器](/wiki/Read-only_memory)中。巴贝奇多年来一直致力于设计，但在大约1840年之后，设计变化似乎很小。这台机器会在40位数字上运行；'mill'(CPU)将有2个主累加器和一些用于特定目的的辅助累加器，而'store'（内存）将保存一千个50位数字。本来会有几个打孔卡读卡器，用于程序和数据;卡片被链接起来，每条链条的运动都是可逆的。机器会执行条件跳转。也会有一种[微编码](/wiki/Microcode)形式：[指令](/wiki/Instruction_(computer_science))的含义取决于金属螺柱在开槽桶中的位置，称为“控制桶”。所设想的机器能够在3秒内完成加法运算，在2-4分钟内完成乘法或除法运算。它由[蒸汽机提供动力](/wiki/Steam_engine)。最后，实际建造的部件不超过几个。
# 1835年
- 在美国，[约瑟夫亨利](/wiki/Joseph_Henry)发明了机电[继电器](/wiki/Relay)。
# 1840年
- 在意大利，[查尔斯·巴贝奇](/wiki/Charles_Babbage)的第一次公开阐述他的分析引擎在[艺术学院阿尔Scienze](/w/index.php?titleAccademia_delle_Scienze&actionedit&redlink1)，[都灵](/wiki/Turin)。
# 1842年
- 在法国，TimoleonMaurel为[Arithmaurel](/wiki/Arithmaurel)申请了专利，这是一种具有非常直观的用户界面的机械计算器，特别适用于乘法和除法，因为只要输入操作数，结果就会显示出来。它于1849年在巴黎举行的法国国家展览会上获得了金牌。不幸的是，其设计的复杂性和脆弱性使其无法生产。
- 在英国，巴贝奇[差分引擎的](/wiki/Difference_engine)建造作为正式项目被取消。成本超支相当可观（花费了17,470英镑，2004年的资金约为1,000,000英镑）。
# 1843年
- 在瑞典，[PerGeorgScheutz](/wiki/Per_Georg_Scheutz)和他的儿子Edvard制作了一个5位数字和带有打印机的[差分引擎](/wiki/Difference_engine)的三阶模型；瑞典政府于1851年同意为他们的下一步发展提供资金。
# 1846年
- 在英国，巴贝奇开始研究改进的[差分引擎](/wiki/Difference_engine)（2号差分引擎），到1849年产生了一套完全执行的计划。这台机器本来可以处理7阶差分和31位数字，但没有人发现要付费才能建造它。1989-1991年，伦敦科学博物馆的一个团队确实根据幸存的计划建造了一个。他们使用现代方法构建组件，但公差并不比Clement所能提供的更好……并且，经过一些修补和细节调试，他们发现机器工作正常。2000年，打印机也完成了。
# 1847年
- 在英国，英国数学家[GeorgeBoole](/wiki/George_Boole)开发了二进制代数（[Booleanalgebra](/wiki/Boolean_algebra_(logic))），它在大约一个世纪后开始广泛应用于二进制计算机的设计和运算。见1939年。
# 1851年
- 在法国，经过30年的发展，[ThomasdeColmar](/wiki/Thomas_de_Colmar)通过开始制造大大简化的[算术计](/wiki/Arithmometer)（发明于1820年），开创了机械计算器行业。除了30年后开始的克隆之外，它是40年来世界上唯一可用的计算机（[DorrE.Felt](/wiki/Dorr_Felt)从1887年到1890年仅售出一百台[计算机](/wiki/Comptometer)和几台[计算机](/wiki/Comptograph)）。它的简单性使其成为迄今为止最可靠的计算器。这是一台大机器（一个20位的算术仪足够长，可以占据桌面的大部分）。尽管该算法直到1915年才生产，但直到二战开始，二十家欧洲公司制造了其设计的改进克隆；他们是Burkhardt、Layton、Saxonia、Gräber、Peerless、Mercedes-Euklid、XxX、Archimedes等等……。
# 1853年
- 在瑞典，令巴贝奇高兴的是，[Scheutzes](/wiki/Per_Georg_Scheutz)完成了第一个全尺寸[差分引擎](/wiki/Difference_engine)，他们称之为制表机。它对15位数字和4阶差分进行运算，并像巴贝奇那样产生打印输出。1859年，伦敦的[BryanDonkin](/wiki/Bryan_Donkin)公司按照相同的设计制造了第二台机器。
# 1856年
- 在美国，第一台制表机（见1853年）由[纽约奥尔巴尼](/wiki/Albany,_New_York)的达德利天文台购买，第二台于1857年由英国政府订购。奥尔巴尼机器被用来制作一套天文台；但是天文台的馆长因为这次奢侈的购买被解雇了，这台机器再也没有认真使用过，最终进了博物馆。第二台机器有很长的使用寿命。
# 1859年
- 在瑞典，[MartinWiberg](/wiki/Martin_Wiberg)生产了一种重新设计[的类似差分引擎的](/wiki/Difference_engine)机器，用于准备利率（1860年首次出版）和[对数](/wiki/Logarithm)表（1875年首次出版）。
# 1866年
- 在英国，第一台实用的逻辑机器（[逻辑算盘](/wiki/Logical_abacus)）是由[威廉·斯坦利·杰文斯](/wiki/William_Stanley_Jevons)(WilliamStanleyJevons)建造的。
# 1871年
- 在英国，巴贝奇制作了[分析引擎](/wiki/Analytical_Engine)磨机和打印机的原型部分。
# 1878年
- 在西班牙，居住在纽约市的RamónVerea发明了一个带有内部乘法表的计算器；这比移动马车或当时的其他数字方法要快得多。然而，他对将其投入生产并不感兴趣。似乎他只是想表明西班牙人可以像美国人一样进行发明。
- 在英国，一个委员会调查了完成[分析引擎](/wiki/Analytical_Engine)的可行性，并得出结论，现在巴贝奇已经死了，这是不可能的。除了极少数人之外，该项目在很大程度上被遗忘了。[霍华德艾肯](/wiki/Howard_Aiken)是一个明显的例外。
# 1884年
- 在美国，芝加哥的DorrFelt开发了他的[Comptometer](/wiki/Comptometer)。这是第一个通过按键输入操作数而不是例如拨入的计算器。这是可行的，因为Felt发明了一种进位机制，其速度足够快，可以在按键从按下状态返回时执行操作。Felt和Tarrant于1887年开始合作制造计算器。
# 1886年
- 在美国，巴尔的摩卫生部首次使用[HermanHollerith](/wiki/Herman_Hollerith)制表系统。
# 1887年
- 在美国，[HermanHollerith](/wiki/Herman_Hollerith)提交了一项[集成制表器](/wiki/Unit_record_equipment#Timeline)的专利申请（1890年获得批准），它可以添加[打孔卡](/wiki/Punched_card)上编码的数字。1889年，陆军外科医生办公室首次记录使用该设备。1896年Hollerith引入了改进模型。
# 1889年
- 在美国，[DorrFelt](/wiki/Dorr_Felt)发明了第一台打印台计算器。
# 1890年
- 在美国瑞典俄罗斯，比算术计更紧凑的乘法计算器进入批量生产。该设计是美国的[FrankS.Baldwin](/wiki/Frank_S._Baldwin)和居住在俄罗斯的瑞典人[WillgodtTheophilOdhner](/wiki/Willgodt_Theophil_Odhner)的独立发明，或多或少是同时发明的。凹槽鼓被“可变齿齿轮”设计所取代：一个带有径向销钉的圆盘，可以从中伸出或缩回。
- 在美国，在[1880年美国人口普查](/wiki/1880_United_States_Census)采取了7年完整的，因为所有的处理一直用手从期刊表来完成。不断增长的人口表明，到[1890年人口普查时](/wiki/1890_United_States_Census)，数据处理所需的时间将比下次人口普查前的10年更长——因此举行了一场竞赛以寻找更好的方法。它是由人口普查部员工[HermanHollerith](/wiki/Herman_Hollerith)赢得的，他后来[创立](/wiki/Tabulating_Machine_Company)了制表机公司，后来成为[IBM](/wiki/International_Business_Machines)。他发明了将数据记录在可以被机器读取的介质上。以前使用机器可读媒体是为了控制（[自动机](/wiki/Automaton)、[钢琴卷](/wiki/Piano_roll)、[织布机）](/wiki/Jacquard_loom),...)，而不是数据。“在使用纸带进行了一些初步试验后，他决定使用[穿孔卡片](/wiki/Punched_card)……”他的机器使用机械[继电器](/wiki/Relay)来增加机械计数器。这种方法用于1890年的人口普查。1880年人口普查的许多变化的净效果：更多的人口、要收集的数据项、人口普查局的人数、预定的出版物以及使用Hollerith的机电制表机，减少了处理人口普查所需的时间从[1880年人口普查的](/wiki/U.S._Census,_1880)八年到1890年人口普查的六年。本发明的灵感来自Hollerith在美国西部旅行期间对铁路列车员的观察;他们用打票的方式对乘客（高大、秃顶、男性）进行了粗略的描述。
# 1891年
- 在美国，圣路易斯的WilliamS.Burroughs于1885年发明了一种类似于Felt的机器（见1884年），但与Comptometer不同的是，它是一种“钥匙设置”机器，仅在拉动曲柄手柄后处理每个数字。这台机器的真正制造始于1891年，尽管Burroughs于1886年创立了他的[美国算术公司](/wiki/American_Arithmometer_Company)（后来成为[Burroughs公司](/wiki/Burroughs_Corporation)，现在称为[Unisys](/wiki/Unisys)）。
# 1899年
- 在日本，[RyōichiYazu](/wiki/Ry%C5%8Dichi_Yazu)开始开发机械计算机（自动算盘）。Ryoichi独立进行了计算机器的研究，在1902年申请专利之前，他用了三年时间完成了他的[二元](/wiki/Bi-quinary_coded_decimal)机械台式计算机器。这是日本第一台成功的机械计算机。
# 1900年
- 在美国，该[标准增加机械公司](/wiki/Standard_Adding_Machine_Company)发布了第一个10键添加机约1900发明者，威廉·霍普金斯，提交于10月4日，1892年10个按键分别在同一行上设置了他的第一个专利。
# 1902年
- 在美国，道尔顿加法机的第一个模型建成。Remington将道尔顿加法机宣传为第一台10键印刷加法机。10个键被设置在两行。到1906年底，已经制造了六台机器。
# 1905年
- 在日本，IchitaroKawaguchi是[交通运输部的](/wiki/Ministry_of_Communications_(Japan))一名工程师，他建造了日本第一台[机电](/wiki/Electromechanical)计算机KawaguchiElectricTabulationMachine，用于将1904年人口统计研究的一些结果制成表格。
# 1906年
- 在英国，查尔斯的儿子亨利·巴贝奇在RWMunro公司的帮助下，用他父亲的[分析](/wiki/Analytical_Engine)机完成了“磨坊”，以证明它是可行的。确实如此。未生产整机。
- 在美国，由LeeDeForest发明的Audion（[真空管](/wiki/Vacuum_tube)或[热离子阀](/wiki/Thermionic_valve)）。
- 在美国，[HermanHollerith](/wiki/Herman_Hollerith)推出了一种带有[插板](/wiki/Plugboard)的制表机，可以重新接线以使机器适应不同的应用。插件板被广泛用于指导机器计算，直到1950年代被[存储的程序](/wiki/Stored_program)取代。
# 1919年
- 在英国，[WilliamHenryEccles](/wiki/William_Henry_Eccles)和[FWJordan](/wiki/F._W._Jordan)发表了第一个[触发器](/wiki/Flip-flop_(electronics))电路设计。
# 1924年
- 在德国，[WaltherBothe](/wiki/Walther_Bothe)建立了一个**AND** [逻辑门](/wiki/Logic_gate)\-[巧合电路](/wiki/Coincidence_circuit)，用于物理实验，他因此获得了1954年的[诺贝尔物理学奖](/wiki/Nobel_Prize)。各种数字电路大量使用这种技术。
# 1928年
- 在美国，[IBM](/wiki/IBM)对带有80列数据和矩形孔的[穿孔卡片](/wiki/Punched_card)进行了标准化。它们被广泛称为IBM卡，主导了数据处理行业近半个世纪。
# 1929年
- 在美国，[西屋](/wiki/Westinghouse_Electric_(1886))AC计算板。直到1960年代一直用于电力传输线模拟的[网络分析仪（交流电源）](/wiki/Network_analyzer_(AC_power))。
# 1930年
- 在美国，[VannevarBush](/wiki/Vannevar_Bush)建造了一个能够求解微分方程的部分电子[微分分析仪](/wiki/Differential_analyser)。
- 在英国，威尔士物理学家[CE韦恩-威廉姆斯](/wiki/C._E._Wynn-Williams)，在[英国剑桥](/wiki/Cambridge,_England)，使用的环[闸流管](/wiki/Thyratron)的管来构造所计数发射的二进制数字计数器[α粒子](/wiki/Alpha_particle)。
# 1931年
- 在奥地利，[哥德尔](/wiki/Kurt_G%C3%B6del)的[维也纳大学](/wiki/Vienna_University)，奥地利，基于算术运算的通用形式语言发表了一篇论文。他用它来编码任意的形式陈述和证明，并表明传统数学等[形式系统](/wiki/Formal_system)要么在某种意义上不一致，要么包含无法证明但真实的陈述。这个结果通常被称为理论计算机科学的基本结果。
- 在美国，[IBM](/wiki/IBM)推出了[IBM601](/wiki/IBM_601)MultiplyingPunch，这是一种机电机器，可以从一张卡上读取两个数字，最长8位，并将他们的产品打到同一张卡上。
# 1934年
- 在日本,苏联,美国，1934年到1937年，[NEC](/wiki/NEC)工程师[AkiraNakashima](/wiki/Akira_Nakashima)、[ClaudeShannon](/wiki/Claude_Shannon)和ViktorShetakov发表了一系列介绍[开关电路理论](/wiki/Switching_circuit_theory)的论文。
- 在美国，[华莱士·埃克特](/wiki/Wallace_Eckert)的[哥伦比亚大学](/wiki/Columbia_University)连接的IBM285制表，一个016仿型冲床和IBM601乘法冲床用[凸轮](/wiki/Cam)\-controlled序器开关，他设计的。组合系统用于自动积分[微分方程](/wiki/Differential_equation)。
# 1936年
- 在英国，[阿兰·图灵](/wiki/Alan_Turing)的[剑桥大学](/wiki/University_of_Cambridge)，英格兰，发表了“可计算数字”一文其重新[哥德尔](/wiki/Kurt_G%C3%B6del)的结果（见相关工作[邱奇](/wiki/Alonzo_Church)）。他的论文解决了著名的“[Entscheidungsproblem](/wiki/Entscheidungsproblem)”，在论文中通过推理（作为一种数学设备）关于一台简单的理论计算机（今天称为[图灵机](/wiki/Turing_machine)）来寻求解决方案。在许多方面，这种设备比哥德尔基于算术的通用形式系统更方便。
# 1937年
- 在美国，纽约市贝尔电话实验室（贝尔实验室）的GeorgeStibitz使用继电器构建了一个演示1位二进制加法器。这是最早的二进制计算机之一，虽然在这个阶段它只是一台演示机；改进继续导致了1940年1月的[复数计算器](/wiki/Complex_Number_Calculator)。
- 在美国，[ClaudeE.Shannon](/wiki/Claude_E._Shannon)在麻省理工学院的硕士论文中发表了一篇关于使用继电器实现符号逻辑的论文。
# 1938年
- 在德国，柏林的KonradZuse完成了“[Z1](/wiki/Z1_(computer))”，这是第一台机械二进制可编程计算机。它基于布尔代数并具有现代机器的一些基本成分，使用二进制系统和浮点运算。Zuse1936年的专利申请(Z23139/GMDNr.005/021)还提出了一种“冯诺依曼”架构（大约在1945年重新发明），其中程序和数据可在存储中修改。该机器最初被称为“V1”，但战后追溯改名，以避免与V-1飞弹混淆.它适用于浮点数（7位指数、16位尾数和符号位）。内存使用滑动金属部件来存储16个这样的数字，运行良好；但算术单元不太成功，偶尔会遇到某些机械工程问题。该程序是从废弃的35毫米电影胶片上打孔中读取的。数据值可以从数字键盘输入，输出显示在电灯上。这台机器不是通用计算机（即[图灵完备](/wiki/Turing_complete)），因为它缺乏循环功能。
# 1939年
- 在美国，[威廉·休利特](/wiki/William_Reddington_Hewlett)和[大卫·帕卡德](/wiki/David_Packard)在加利福尼亚州帕洛阿尔托帕卡德的车库中成立了[惠普公司](/wiki/Hewlett-Packard_Company)，初期投资为538美元（相当于2020年的9,891美元）；这被认为是硅谷的象征性成立。惠普将发展成为当今世界上最大的科技公司之一。
- 在德国，[HelmutSchreyer](/wiki/Helmut_Schreyer)使用真空管完成了一个原型10位加法器，以及一个使用霓虹灯的原型存储器。
- ## 11月
	- 在美国，[JohnVincentAtanasoff](/wiki/John_Vincent_Atanasoff)和爱荷华州立大学（现爱荷华州立大学）的研究生[CliffordBerry](/wiki/Clifford_Berry)完成了一个原型16位加法器。这是第一台使用真空管进行计算的机器。
# 1940年
- 在美国，在[贝尔实验室](/wiki/Bell_Labs)，塞缪尔·威廉姆斯和[乔治·斯蒂比茨](/wiki/George_Stibitz)完成了一个可以对[复数](/wiki/Complex_number)进行运算的计算器，并将其命名为“[复数计算器”。](/wiki/Complex_Number_Calculator)';它后来被称为“I型继电器计算器”。它使用电话交换部件进行逻辑：450个继电器和10个纵横开关。数字以“加3BCD”表示；也就是说，对于每个十进制数字，0用二进制0011表示，1用0100表示，依此类推，直到1100表示9；该方案比直接BCD需要更少的继电器。计算器并没有要求用户到机器前使用它，而是在建筑物的不同地方以电传打字机的形式提供了三个远程键盘。一次只能使用一个，并且输出会自动显示在同一个上。1940年9月9日，在新罕布什尔州汉诺威的[达特茅斯学院](/wiki/Dartmouth_College)安装了一台电传打字机，连接到纽约，参加会议的人可以远程使用机器。
- 在德国，[KonradZuse](/wiki/Konrad_Zuse)完成了“[Z2](/wiki/Z2_(computer))”（原为“V2”），它将Z1现有的机械记忆单元与使用继电器逻辑的新算术单元相结合。与Z1一样，Z2缺乏循环功能。该计划在1939年起草Zuse时中断了一年，但在他获释后继续进行。1940年，ZuseZ2在柏林阿德勒斯霍夫向德国航空研究所（“德国航空实验室”）的观众展示。
# 1941年
- ## 5月
	- ### 11日
		- 在德国，现在，在DVL（德国航空研究所）的有限支持下，[KonradZuse](/wiki/Konrad_Zuse)完成了“**[Z3](/wiki/Z3_(computer))”**（原为“V3”）：第一台可操作的可编程计算机。对[CharlesBabbage](/wiki/Charles_Babbage)的非功能性设备的一项重大改进是使用[Leibniz](/wiki/Gottfried_Leibniz)的二进制系统（巴贝奇和其他人尝试构建十进制可编程计算机失败）。Zuse的机器还具有浮点数，具有7位指数、14位尾数（除非数字为0，否则自动带有“1”位前缀）和一个符号位。内存保存了64个这样的单词，因此需要1400多个继电器；算术和控制单元有1200多个。它还具有并行加法器。程序、输入和输出的实现与上述Z1相同。虽然条件跳转不可用，但事实证明，Zuse的Z3原则上能够作为通用计算机运行。机器每秒可以做3-4次加法，乘法需要3-5秒。Z3于1943年在盟军轰炸柏林期间被摧毁，对美国和英国的计算机技术没有影响。
# 1942年
- 在美国，Atanasoff和Berry完成了一个用于求解联立线性方程组的专用计算器，后来称为“ABC”（“[Atanasoff–Berry计算机](/wiki/Atanasoff%E2%80%93Berry_Computer)”）。它有60个50位字的存储器，以[电容器](/wiki/Capacitor)形式（带有刷新电路——第一个再生存储器）安装在两个转鼓上。时钟速度为60Hz，加法需要1秒。对于辅助存储器，它使用穿孔卡片，由用户移动。这些孔实际上并没有在卡片上打孔，而是被烧毁了。打孔卡系统的错误率从未降低到0.001%以上，这是不够的。美国参战后，阿塔纳索夫离开了爱荷华州，结束了他在数字计算机方面的工作。
- 在德国，[HelmutHölzer](/wiki/Helmut_H%C3%B6lzer)建造了一台[模拟计算机](/wiki/Analog_computer)来计算和模拟 [V-2火箭](/wiki/V-2_rocket)弹道。
- 在德国，[KonradZuse](/wiki/Konrad_Zuse)开发了S1，这是世界上第一台过程计算机，[Henschel](/wiki/Henschel)使用它来测量机翼表面。
# 1943年
- ## 4月
	- 在英国，[MaxNewman](/wiki/Max_Newman)、[CEWynn-Williams](/wiki/C._E._Wynn-Williams)和他们在英国[布莱切利布莱切利公园](/wiki/Bletchley_Park)秘密政府密码和密码学校（“X站”）的团队完成了“[希思·罗宾逊](/wiki/Heath_Robinson_(codebreaking_machine))”。这是用于破译密码的专用计数机，不是通用计算器或计算机，而是使用电子和继电器逻辑组合的逻辑设备。它以每秒2000个字符的速度从两个闭合的纸带环中以光学方式读取数据。它很重要，因为它是Colossus的先驱。纽曼在[剑桥大学](/wiki/University_of_Cambridge)认识图灵（图灵是纽曼的学生），并且是第一个看到图灵1936年论文草稿的人。 [希思·罗宾逊](/wiki/Heath_Robinson)是一位以画滑稽机器而闻名的英国[漫画家](/wiki/Cartoonist)的名字，比如美国的[鲁布·戈德堡](/wiki/Rube_Goldberg)。该系列中后来的两台机器以伦敦商店的名字命名，名字中带有“Robinson”。
- ## 9月
	- 在美国，Williams和Stibitz完成了“继电器内插器”，后来称为“[模型II继电器计算器](/wiki/Bell_Labs#Calculators)”。这是一个可编程计算器；同样，程序和数据是从纸带中读取的。一个创新功能是，为了提高可靠性（错误检测/自检），数字以[双五进制](/wiki/Bi-quinary_coded_decimal)格式表示，每个数字使用七个继电器，其中两个应该“打开”：0100001代表0,0100010为1，依此类推，直到1010000为9。本系列中的一些后期机器将使用二元表示法表示浮点数的数字。
- ## 12月
	- 在英国，马克1[巨像](/wiki/Colossus_computer)完成，由[汤米·弗劳尔斯](/wiki/Tommy_Flowers)在[邮局研究实验室](/wiki/Post_Office_Research_Station)在伦敦，以协助在德国的开裂[洛伦兹SZ42密码](/wiki/Lorenz_cipher)在布莱切利公园。它是一个二进制数字机器，包含1500个[真空管（阀门）](/wiki/Vacuum_tube)，并将可编程逻辑功能应用于字符流，以每秒5000个字符的速度从[穿孔纸带](/wiki/Punched_tape)循环中读取和重新读取。它有501[位](/wiki/Bit)内存，程序设置在开关和插头面板上。Colossus在二战期间在[BletchleyPark](/wiki/Bletchley_Park)使用-作为生产力较低的HeathRobinson机器的后续产品。
# 1944年
- ## 6月
	- 在英国，第一个马克2巨像被委托。它是Mark1机器的开发，包含2400个真空管。它有五个相同的并行处理器，由一个[移位寄存器提供](/wiki/Shift_register)，每秒可处理25,000个字符。Colossus可以评估各种[布尔代数](/wiki/Boolean_algebra)函数，以帮助建立LorenzSZ42机器的转子设置。到1945年5月欧洲战争结束时，布莱切利公园使用了10辆Mark2Colossi。然后，除了两台机器外，其他所有机器都被拆成小部件，无法推断它们的用途，以保持工作的保密性。其余两个在1960年代在[GCHQCheltenham](/wiki/GCHQ)拆除。
- ## 8月
	- ### 7日
		- 在美国，在[IBM](/wiki/IBM)自动序列控制计算器被移交给[哈佛大学](/wiki/Harvard_University)，调用它的[马克一号](/wiki/Harvard_Mark_I)。它由[HowardAiken](/wiki/Howard_Aiken)和他的团队设计，由IBM资助和建造——它成为第二台程控机器（在[KonradZuse](/wiki/Konrad_Zuse)之后）。整台机器长51英尺（16m），重5（短）吨（4.5吨），包含750,000个零件。它使用3304个机电继电器作为通断开关，有72个累加器（每个都有自己的算术单元），以及一个容量为23位加号的机械寄存器。算术是定点和十进制的，带有[控制面板](/wiki/Control_panel_(computer))设置确定小数位数。输入输出设备包括读卡器、打卡机、纸带阅读器和打字机。有60组旋转开关，每组都可以用作常量寄存器——一种机械[只读存储器](/wiki/Read-only_memory)。该程序是从一张纸带中读取的；数据可以从其他磁带、读卡器或常量寄存器中读取。条件跳转不可用。然而，在后来的几年里，机器被修改为支持程序的多个纸带阅读器，从一个到另一个的转移是有条件的，就像一个有条件的子程序调用。另一个附加功能允许提供可从磁带调用的插板有线子程序。用于为美国海军创建弹道表.。
# 1945年
- 在德国，[KonradZuse](/wiki/Konrad_Zuse)开发了[Plankalkül](/wiki/Plankalk%C3%BCl)，这是第一种高级编程语言。他还在三月份展示了[Z4](/wiki/Z4_(computer))。
- 在美国，[VannevarBush](/wiki/Vannevar_Bush)开发了[memex](/wiki/Memex)理论，这是一种链接到图书和电影库的[超文本](/wiki/Hypertext)设备。
- 在美国，[约翰·冯·诺依曼](/wiki/John_von_Neumann)起草了一份报告，描述了最终构建为[EDVAC](/wiki/EDVAC)（电子离散变量自动计算机）的未来计算机。[EDVAC报告的初稿](/wiki/First_Draft_of_a_Report_on_the_EDVAC)包括对[存储程序计算机](/wiki/Stored-program_computer)设计的首次公开描述，从而产生了[冯诺依曼体系结构](/wiki/Von_Neumann_architecture)这一术语。它直接或间接地影响了几乎所有后续项目，尤其是[EDSAC](/wiki/EDSAC)。设计团队包括[JohnW.Mauchly](/wiki/John_W._Mauchly)和[J.PresperEckert](/wiki/J._Presper_Eckert)。
# 1946年
- 在英国，该[轨迹球](/wiki/Trackball)被发明的名为雷达标绘系统的一部分，[综合显示系统](/wiki/Comprehensive_Display_System)由（CDS）[拉尔夫·本杰明](/wiki/Ralph_Benjamin)在英国工作时，[皇家海军](/wiki/Royal_Navy)科学的服务。本杰明的项目使用[模拟计算机](/wiki/Analog_computer)根据用户提供的几个初始输入点计算目标飞机的未来位置与[操纵杆](/wiki/Joystick)。本杰明认为需要一个更优雅输入装置和发明了一种[球跟踪器](/wiki/Ball_tracker)系统，称为[滚子球](/wiki/Roller_ball_(pointing_device))为此目的，在1946年该设备于1947年获得专利，但只制造了一个原型并且该设备在军事之外被保密。
- ## 2月
	- ### 14日
		- 在美国，[ENIAC](/wiki/ENIAC)（ElectronicNumericalIntegratorandComputer）：第一台全电子、真空管、数字、程控计算机亮相，尽管它于1946年11月9日因翻新和内存升级而关闭，并转移到AberdeenProving1947年，马里兰州地面。1943年，JohnW.Mauchly和J.PresperEckert在美国[弹道研究实验室](/wiki/Ballistic_Research_Laboratory)开始开发。它重30吨，包含18,000个真空管，消耗约160千瓦的电力。它每秒可以进行5,000次基本计算。它被用于计算弹道轨迹和测试氢弹背后的理论。
	- ### 19日
		- 在英国，[ACE](/wiki/Automatic_Computing_Engine)（自动计算引擎）：[AlanTuring](/wiki/Alan_Turing)向[国家物理实验室(NPL)](/wiki/National_Physical_Laboratory,_UK)执行委员会提交了一份详细的论文，给出了第一个合理完整的存储程序计算机设计。然而，由于他在[布莱切利公园的](/wiki/Bletchley_Park)战时工作严格且长期保密，他被禁止（签署了《[官方保密法》](/wiki/Official_Secrets_Act)）解释说他知道他的想法可以在电子设备中实现。
# 1947年
- 在美国，[HowardAiken](/wiki/Howard_Aiken)完成了[哈佛MarkII](/wiki/Harvard_Mark_II)。
- 在美国，在[美国计算机协会](/wiki/Association_for_Computing_Machinery)（ACM），建立了作为世界上第一个科学和教育计算社会。直到今天，会员人数仍约为78,000人。其总部位于纽约市。
- ## 9月
	- 在英国，第一个开发[汇编语言](/wiki/Assembly_language)由[凯瑟琳·布思](/wiki/Kathleen_Booth)在[伦敦大学伯克贝克学院，](/wiki/Birkbeck,_University_of_London)在与工作[约翰·冯·诺伊曼](/wiki/John_von_Neumann)和[赫尔曼·戈德斯坦](/wiki/Herman_Goldstine)在[高等研究院](/wiki/Institute_for_Advanced_Study)。
- ## 12月
	- ### 16日
		- 在美国，所述的本发明[晶体管](/wiki/Transistor)在[贝尔实验室](/wiki/Bell_Laboratories)，USA，通过[威廉B.肖克莱](/wiki/William_B._Shockley)，[约翰巴丁](/wiki/John_Bardeen)和[沃尔特布拉顿](/wiki/Walter_Brattain)。
# 1948年
- 在美国，[ANACOM](/wiki/Network_analyzer_(AC_power)#Anacom)从[西屋](/wiki/Westinghouse_Electric_(1886))是一个AC-带电电气[模拟计算机](/wiki/Analog_computer)使用，直到20世纪90年代早期在机械和结构的设计，流体，以及各种瞬态问题的问题的系统。
- 在美国，[IBM](/wiki/IBM)推出了“[604](/wiki/IBM_604)”，这是第一台具有[现场可更换单元](/wiki/Field_Replaceable_Unit)(FRU)功能的机器，它减少了停机时间，因为整个可插拔单元可以简单地更换而不是故障排除。
- 在1948，第一台[Curta](/wiki/Curta)手持机械计算器售出。Curta以11位十进制精度计算输入操作数，最多8位十进制数。Curta大约有手持胡椒研磨机那么大。
- ## 1月
	- ### 27日
		- 在美国，[IBM](/wiki/IBM)完成了[SSEC](/wiki/IBM_SSEC)（选择性序列电子计算器）。它是第一台修改存储程序的计算机。“大约使用了1300个真空管来构建算术单元和8个非常高速的寄存器，而在控制结构中使用了23000个继电器和150个较慢的存储器寄存器。”。
- ## 5月
	- ### 12日
		- 在英国，在[伯克贝克ARC](/wiki/APEXC)，前三机在开发的[伯克贝克学院，伦敦大学](/wiki/Birkbeck,_University_of_London)的[安德鲁·布斯](/wiki/Andrew_Donald_Booth)和[凯瑟琳展位](/wiki/Kathleen_Booth)，网上正式宣告在这个日期。控制完全是机电式的，记忆是基于[旋转磁鼓](/wiki/Drum_memory)。这是现存的第一个转鼓存储设备。
- ## 6月
	- ### 21日
		- 在英国，在[曼彻斯特婴儿](/wiki/Manchester_Baby)在内置[曼彻斯特大学](/wiki/University_of_Manchester)。它在这一天运行了第一个程序。它是第一台像现代计算机一样将其程序和数据存储在[RAM中的](/wiki/Random-access_memory)计算机。到1949年，“婴儿”已经长大，并获得了一个[磁鼓](/wiki/Drum_memory)以进行更[持久的存储](/wiki/Computer_storage)，它成为[曼彻斯特马克1](/wiki/Manchester_Mark_1)。
# 1949年
- 在澳大利亚，CSIRMkI（后称为[CSIRAC](/wiki/CSIRAC)）是澳大利亚的第一台计算机，运行了它的第一个测试程序。它是一种基于真空管的电子通用计算机。它的主存储器将数据作为一系列声波脉冲存储在5英尺（1.5米）长的充满水银的管子中。
- 在英国，[MONIAC](/wiki/MONIAC)（货币国民收入模拟计算机）也称为菲利普斯液压计算机，创建于1949年，用于模拟英国的国民经济过程。MONIAC由一系列透明塑料罐和管道组成。人们认为制造了十二到十四台机器。
- ## 3月
	- 在美国，[JohnPresperEckert](/wiki/John_Presper_Eckert)和[JohnWilliamMauchly](/wiki/John_William_Mauchly)为Northrop构建了[BINAC](/wiki/BINAC)。
- ## 5月
	- ### 6日
		- 在英国，这被认为是现代计算的诞生日。 [MauriceWilkes](/wiki/Maurice_Wilkes)和[剑桥大学的](/wiki/University_of_Cambridge)一个团队在[EDSAC](/wiki/EDSAC)计算机上执行了第一个存储程序，该程序使用纸带输入-输出。基于[约翰·冯·诺依曼](/wiki/John_von_Neumann)关于存储程序计算机的想法，EDSAC是第一台完整的、功能齐全的冯·诺依曼架构计算机。
- ## 10月
	- 在英国，在[曼彻斯特马克1种](/wiki/Manchester_Mark_1)最终规格完成;值得注意的是，这台机器是第一台使用等效基址/[索引寄存器的](/wiki/Index_register)计算机，该功能直到1955年左右的第二代才进入通用计算机体系结构。
